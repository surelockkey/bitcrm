# TR-25: Inventory - Stocktaking

**Epic:** Inventory Management System
**Component:** Physical Inventory Counts & Adjustments
**Version:** 1.0
**Last Updated:** October 31, 2025

---

## 1. Overview

This story covers physical inventory counting (stocktaking), discrepancy detection, inventory adjustments, and cycle count scheduling. Stocktaking ensures system inventory matches physical reality.

**Key Objectives:**
- Support full and partial (cycle) counts
- Blind count methodology to prevent bias
- Barcode-driven counting for accuracy
- Automatic discrepancy detection
- Adjustment approval workflow
- Real-time variance reporting
- Mobile-first experience for technicians
- Desktop management for warehouse
- Complete audit trail

---

## 2. Stocktaking Types

### 2.1. Full Count

Complete count of all inventory in a location.

**When used:**
- End of month/quarter/year
- New warehouse setup
- Major discrepancies found
- Audit requirements
- Technician onboarding/offboarding

**Frequency:**
- Central Warehouse: Monthly
- Technician Containers: Quarterly or as needed

### 2.2. Cycle Count

Partial count of specific products or categories.

**When used:**
- High-value items (weekly/biweekly)
- High-turnover items
- Items with frequent discrepancies
- Random spot checks

**Frequency:**
- High-value items (>$50): Weekly
- Fast-moving items: Biweekly
- Random 10% of SKUs: Weekly

### 2.3. Spot Check

Ad-hoc count of specific items.

**When used:**
- Suspected theft/loss
- Customer complaint about parts
- Before/after major job
- Transfer discrepancy investigation

## 3. Blind Count Methodology

**Blind Count:** Counter doesn't see system quantity, preventing bias.

```
Traditional Count (BAD):
  System says: 45 units
  Counter thinks: "Looks about right, I'll put 45"
  Result: Errors not caught

Blind Count (GOOD):
  System says: [hidden]
  Counter counts: Actually counts everything
  Counter enters: 43 units
  System shows: Expected 45, Found 43, Variance: -2
  Result: Accurate count, discrepancies found
```

### 3.1. Blind Count Process

```
1. Start stocktaking session
   ↓
2. System provides list of items to count
   WITHOUT showing expected quantities
   ↓
3. Counter physically counts each item
   ↓
4. Counter enters actual count
   ↓
5. System compares with expected quantity
   ↓
6. System shows variances
   ↓
7. Counter reviews variances
   ↓
8. Counter can recount items with large variances
   ↓
9. Submit stocktaking session
   ↓
10. Manager reviews and approves adjustments
```

### 10.2. Stocktaking API

```php
// File: /local/api/inventory/stocktaking.php

namespace Local\Inventory;

class StocktakingManager {

    /**
     * Create new stocktaking session
     * POST /api/inventory/stocktaking/create
     */
    public static function create($data) {
        global $DB;
        $userId = $GLOBALS['USER']->GetID();

        $stocktakingNumber = self::generateStocktakingNumber();
        $storeId = $data['store_id'];
        $type = $data['type']; // 'full', 'cycle', 'spot'
        $isBlind = $data['is_blind'] ?? true;

        // Create session
        $DB->Query("
            INSERT INTO b_inventory_stocktaking
            (STOCKTAKING_NUMBER, STORE_ID, STOCKTAKING_TYPE, STATUS, IS_BLIND_COUNT,
             STARTED_BY, DATE_CREATE, DATE_START)
            VALUES
            ('{$stocktakingNumber}', {$storeId}, '{$type}', 'in_progress', " . ($isBlind ? 'TRUE' : 'FALSE') . ",
             {$userId}, NOW(), NOW())
        ");

        $stocktakingId = $DB->LastID();

        // Get items to count
        if ($type === 'full') {
            $items = self::getAllStoreProducts($storeId);
        } else {
            $items = self::getSelectedProducts($data['product_ids']);
        }

        // Add items to session
        foreach ($items as $item) {
            self::addItemToStocktaking($stocktakingId, $item);
        }

        // Update totals
        $DB->Query("
            UPDATE b_inventory_stocktaking
            SET TOTAL_ITEMS = " . count($items) . "
            WHERE ID = {$stocktakingId}
        ");

        return [
            'stocktaking_id' => $stocktakingId,
            'stocktaking_number' => $stocktakingNumber,
            'total_items' => count($items),
            'is_blind' => $isBlind
        ];
    }

    /**
     * Record count for an item
     * POST /api/inventory/stocktaking/{id}/count-item
     */
    public static function countItem($stocktakingId, $productId, $countedQuantity, $note = null) {
        global $DB;

        // Get stocktaking item
        $item = $DB->Query("
            SELECT * FROM b_inventory_stocktaking_item
            WHERE STOCKTAKING_ID = {$stocktakingId} AND PRODUCT_ID = {$productId}
        ")->Fetch();

        if (!$item) {
            throw new \Exception("Product not in this stocktaking session");
        }

        // Calculate variance
        $variance = $countedQuantity - $item['QUANTITY_EXPECTED'];

        // Get product cost for value variance
        $product = \CIBlockElement::GetByID($productId)->Fetch();
        $costTech = $product['UF_COST_TECH'];
        $valueVariance = $variance * $costTech;

        // Update item
        $DB->Query("
            UPDATE b_inventory_stocktaking_item
            SET QUANTITY_COUNTED = {$countedQuantity},
                QUANTITY_VARIANCE = {$variance},
                VALUE_VARIANCE = {$valueVariance},
                STATUS = 'counted',
                COUNTED_AT = NOW(),
                TECHNICIAN_NOTE = " . ($note ? "'{$DB->ForSql($note)}'" : "NULL") . "
            WHERE ID = {$item['ID']}
        ");

        // Update session counts
        $DB->Query("
            UPDATE b_inventory_stocktaking
            SET COUNTED_ITEMS = COUNTED_ITEMS + 1,
                MATCHED_ITEMS = MATCHED_ITEMS + " . ($variance == 0 ? '1' : '0') . ",
                VARIANCE_ITEMS = VARIANCE_ITEMS + " . ($variance != 0 ? '1' : '0') . ",
                TOTAL_VARIANCE_VALUE = TOTAL_VARIANCE_VALUE + {$valueVariance}
            WHERE ID = {$stocktakingId}
        ");

        return [
            'item_id' => $item['ID'],
            'quantity_expected' => $item['QUANTITY_EXPECTED'],
            'quantity_counted' => $countedQuantity,
            'variance' => $variance,
            'value_variance' => $valueVariance,
            'has_variance' => $variance != 0
        ];
    }

    /**
     * Complete stocktaking (submit for approval)
     * POST /api/inventory/stocktaking/{id}/complete
     */
    public static function complete($stocktakingId, $notes = null) {
        global $DB;
        $userId = $GLOBALS['USER']->GetID();

        // Update status
        $DB->Query("
            UPDATE b_inventory_stocktaking
            SET STATUS = 'pending_approval',
                COMPLETED_BY = {$userId},
                DATE_COMPLETE = NOW(),
                NOTES = " . ($notes ? "'{$DB->ForSql($notes)}'" : "NULL") . "
            WHERE ID = {$stocktakingId}
        ");

        // Get summary
        $session = $DB->Query("SELECT * FROM b_inventory_stocktaking WHERE ID = {$stocktakingId}")->Fetch();

        // Notify manager
        self::notifyPendingApproval($stocktakingId);

        return [
            'stocktaking_id' => $stocktakingId,
            'status' => 'pending_approval',
            'total_items' => $session['TOTAL_ITEMS'],
            'counted_items' => $session['COUNTED_ITEMS'],
            'variance_items' => $session['VARIANCE_ITEMS'],
            'total_variance_value' => $session['TOTAL_VARIANCE_VALUE']
        ];
    }

    /**
     * Approve stocktaking and apply adjustments
     * POST /api/inventory/stocktaking/{id}/approve
     */
    public static function approve($stocktakingId, $decisions, $approvalNotes = null) {
        global $DB;
        $userId = $GLOBALS['USER']->GetID();

        // Get session
        $session = $DB->Query("SELECT * FROM b_inventory_stocktaking WHERE ID = {$stocktakingId}")->Fetch();

        if ($session['STATUS'] !== 'pending_approval') {
            throw new \Exception("Stocktaking is not pending approval");
        }

        // Get variance items
        $items = $DB->Query("
            SELECT * FROM b_inventory_stocktaking_item
            WHERE STOCKTAKING_ID = {$stocktakingId} AND QUANTITY_VARIANCE != 0
        ")->fetchAll();

        foreach ($items as $item) {
            $decision = $decisions[$item['ID']] ?? 'approve';

            if ($decision === 'approve') {
                // Create adjustment
                self::createAdjustment($stocktakingId, $item, $userId);
            }
        }

        // Update session
        $DB->Query("
            UPDATE b_inventory_stocktaking
            SET STATUS = 'approved',
                APPROVED_BY = {$userId},
                DATE_APPROVED = NOW(),
                APPROVAL_NOTES = " . ($approvalNotes ? "'{$DB->ForSql($approvalNotes)}'" : "NULL") . "
            WHERE ID = {$stocktakingId}
        ");

        // Notify technician
        self::notifyApproved($stocktakingId);

        return ['status' => 'approved'];
    }

    /**
     * Create adjustment from stocktaking variance
     */
    private static function createAdjustment($stocktakingId, $item, $approverId) {
        global $DB;

        $adjustmentNumber = self::generateAdjustmentNumber();
        $adjustmentType = $item['QUANTITY_VARIANCE'] > 0 ? 'increase' : 'decrease';

        // Create adjustment record
        $DB->Query("
            INSERT INTO b_inventory_adjustment
            (ADJUSTMENT_NUMBER, STOCKTAKING_ID, STOCKTAKING_ITEM_ID, PRODUCT_ID, STORE_ID,
             QUANTITY_BEFORE, QUANTITY_COUNTED, QUANTITY_ADJUSTED, VALUE_ADJUSTED,
             ADJUSTMENT_TYPE, REASON, TECHNICIAN_NOTE, APPROVED_BY, DATE_CREATE, DATE_APPROVED)
            SELECT '{$adjustmentNumber}', {$stocktakingId}, {$item['ID']}, {$item['PRODUCT_ID']},
                   st.STORE_ID, {$item['QUANTITY_EXPECTED']}, {$item['QUANTITY_COUNTED']},
                   {$item['QUANTITY_VARIANCE']}, {$item['VALUE_VARIANCE']},
                   '{$adjustmentType}', 'stocktaking_variance',
                   " . ($item['TECHNICIAN_NOTE'] ? "'{$DB->ForSql($item['TECHNICIAN_NOTE'])}'" : "NULL") . ",
                   {$approverId}, NOW(), NOW()
            FROM b_inventory_stocktaking st
            WHERE st.ID = {$stocktakingId}
        ");

        $adjustmentId = $DB->LastID();

        // Apply inventory adjustment
        self::applyInventoryAdjustment($item, $adjustmentId);

        return $adjustmentId;
    }

    /**
     * Apply inventory adjustment to store
     */
    private static function applyInventoryAdjustment($item, $adjustmentId) {
        global $DB;

        // Get store ID from session
        $session = $DB->Query("
            SELECT STORE_ID FROM b_inventory_stocktaking WHERE ID = {$item['STOCKTAKING_ID']}
        ")->Fetch();

        $storeId = $session['STORE_ID'];
        $productId = $item['PRODUCT_ID'];
        $adjustment = $item['QUANTITY_VARIANCE'];

        // Update store stock
        $stock = \Bitrix\Catalog\StoreProductTable::getList([
            'filter' => ['STORE_ID' => $storeId, 'PRODUCT_ID' => $productId],
            'select' => ['ID', 'AMOUNT']
        ])->fetch();

        if ($stock) {
            \Bitrix\Catalog\StoreProductTable::update($stock['ID'], [
                'AMOUNT' => $stock['AMOUNT'] + $adjustment
            ]);
        }

        // Create movement record
        $movementId = \Local\Inventory\DealInventoryManager::createMovementRecord([
            'product_id' => $productId,
            'source_store_id' => $adjustment < 0 ? $storeId : null,
            'dest_store_id' => $adjustment > 0 ? $storeId : null,
            'dest_type' => 'adjustment',
            'quantity' => abs($adjustment),
            'movement_type' => 'adjustment',
            'reference_type' => 'stocktaking',
            'reference_id' => $item['STOCKTAKING_ID'],
            'notes' => "Stocktaking adjustment: Expected {$item['QUANTITY_EXPECTED']}, Counted {$item['QUANTITY_COUNTED']}",
            'moved_by' => $GLOBALS['USER']->GetID()
        ]);

        // Link movement to adjustment
        $DB->Query("
            UPDATE b_inventory_adjustment
            SET MOVEMENT_ID = {$movementId}
            WHERE ID = {$adjustmentId}
        ");
    }

    /**
     * Helper: Generate stocktaking number
     */
    private static function generateStocktakingNumber() {
        global $DB;
        $year = date('Y');
        $result = $DB->Query("
            SELECT MAX(CAST(SUBSTRING(STOCKTAKING_NUMBER, 9) AS UNSIGNED)) as max_num
            FROM b_inventory_stocktaking
            WHERE STOCKTAKING_NUMBER LIKE 'ST-{$year}-%'
        ")->Fetch();
        $nextNum = ($result['max_num'] ?? 0) + 1;
        return sprintf('ST-%s-%05d', $year, $nextNum);
    }

    /**
     * Helper: Generate adjustment number
     */
    private static function generateAdjustmentNumber() {
        global $DB;
        $year = date('Y');
        $result = $DB->Query("
            SELECT MAX(CAST(SUBSTRING(ADJUSTMENT_NUMBER, 10) AS UNSIGNED)) as max_num
            FROM b_inventory_adjustment
            WHERE ADJUSTMENT_NUMBER LIKE 'ADJ-{$year}-%'
        ")->Fetch();
        $nextNum = ($result['max_num'] ?? 0) + 1;
        return sprintf('ADJ-%s-%05d', $year, $nextNum);
    }
}
```
