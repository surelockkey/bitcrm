## 1. System Overview

### 1.1. Purpose

Implement automated late payment fee calculation and application for overdue invoices in Bitrix24. The system automatically detects overdue invoices and applies configurable late fees based on state-specific regulations and business policies.

**Key Features:**
- Automated daily checks for overdue invoices
- State-specific late fee rates, grace periods, and maximum caps
- Real-time fee calculation when client opens invoice portal
- Automatic invoice total updates
- Timeline tracking for all fee applications
- Manager notifications for high-value overdue invoices

### 1.2. Business Requirements

**Why Late Payment Fees?**

**Cash Flow Management:**
- Incentivizes timely payments
- Compensates for delayed cash flow
- Industry standard practice

**Legal Compliance:**
- Each state has different regulations
- Some states cap maximum late fee amounts
- Grace periods vary by state law

**Customer Relations:**
- Transparent fee disclosure
- Automatic calculation (no manual intervention)
- Fair daily proration

## 3. Late Fee Calculation Logic

### 3.1. Calculation Formula

**Base Formula:**

```
Late Fee = Invoice Amount × (Rate / 100) × (Chargeable Days / 30)

Where:
  Invoice Amount = Subtotal + Tax (excluding existing late fees)
  Rate = State-specific rate (% per month)
  Chargeable Days = Days Overdue - Grace Period

If Maximum Fee is set:
  Late Fee = min(Calculated Fee, Maximum Fee)
```

**Example 1: Texas (1.5%, 5-day grace, no cap)**

```
Invoice Amount: $500.00
Due Date: Oct 31, 2025
Today: Nov 8, 2025

Days Overdue: 8 days
Grace Period: 5 days
Chargeable Days: 8 - 5 = 3 days

Monthly Fee: $500 × 1.5% = $7.50
Daily Fee: $7.50 / 30 = $0.25
Total Late Fee: $0.25 × 3 = $0.75

New Invoice Total: $500.00 + $0.75 = $500.75
```

**Example 2: New York (1.5%, 0-day grace, $50 cap)**

```
Invoice Amount: $5,000.00
Due Date: Oct 31, 2025
Today: Nov 15, 2025

Days Overdue: 15 days
Grace Period: 0 days
Chargeable Days: 15 days

Monthly Fee: $5,000 × 1.5% = $75.00
Daily Fee: $75.00 / 30 = $2.50
Calculated Fee: $2.50 × 15 = $37.50

Maximum Fee: $50.00
Total Late Fee: min($37.50, $50.00) = $37.50

New Invoice Total: $5,000.00 + $37.50 = $5,037.50
```

**Example 3: Grace Period Protection**

```
Invoice Amount: $300.00
Due Date: Nov 1, 2025
Today: Nov 3, 2025
Grace Period: 5 days

Days Overdue: 2 days
Chargeable Days: 2 - 5 = -3 (negative)

Late Fee: $0.00 (within grace period)

Invoice Total: $300.00 (unchanged)
```

### 3.2. Technical Implementation

**File:** `/local/php_interface/include/late_fee_calculator.php`

```php
<?php
namespace Local\Payment;

use Bitrix\Main\Type\DateTime;

class LateFeeCalculator {

    /**
     * Calculate late fee for an invoice
     *
     * @param int $dealId Deal ID
     * @return array ['fee' => float, 'days_overdue' => int, 'rate' => float, 'capped' => bool]
     */
    public static function calculateLateFee($dealId) {
        $deal = \CCrmDeal::GetByID($dealId);
        if (!$deal) {
            throw new \Exception('Deal not found');
        }

        // Get invoice details
        $dueDate = $deal['UF_CRM_INVOICE_DUE_DATE'];
        $paymentStatus = $deal['UF_CRM_PAYMENT_STATUS'];
        $terms = $deal['UF_CRM_TERMS_OF_PAYMENT'];

        // Skip if not applicable
        if (!self::shouldApplyLateFee($dueDate, $paymentStatus, $terms)) {
            return ['fee' => 0, 'days_overdue' => 0, 'rate' => 0, 'capped' => false];
        }

        // Calculate days overdue
        $dueDateTime = strtotime($dueDate);
        $today = time();
        $daysOverdue = floor(($today - $dueDateTime) / 86400);

        if ($daysOverdue <= 0) {
            return ['fee' => 0, 'days_overdue' => 0, 'rate' => 0, 'capped' => false];
        }

        // Get state configuration
        $state = self::getStateFromDeal($deal);
        $config = self::getLateFeeConfig($state);

        // Calculate chargeable days
        $chargeableDays = $daysOverdue - $config['grace_period'];

        if ($chargeableDays <= 0) {
            return ['fee' => 0, 'days_overdue' => $daysOverdue, 'rate' => $config['rate'], 'capped' => false];
        }

        // Get invoice amount (excluding existing late fees)
        $invoiceAmount = self::getInvoiceAmount($dealId);

        // Calculate fee
        $monthlyRate = $config['rate'] / 100;
        $dailyRate = $monthlyRate / 30;
        $calculatedFee = $invoiceAmount * $dailyRate * $chargeableDays;

        // Apply maximum cap if set
        $capped = false;
        if ($config['max_fee'] !== null && $calculatedFee > $config['max_fee']) {
            $calculatedFee = $config['max_fee'];
            $capped = true;
        }

        // Round to 2 decimal places
        $fee = round($calculatedFee, 2);

        return [
            'fee' => $fee,
            'days_overdue' => $daysOverdue,
            'rate' => $config['rate'],
            'capped' => $capped,
            'max_fee' => $config['max_fee'],
            'grace_period' => $config['grace_period']
        ];
    }

    /**
     * Check if late fee should be applied
     */
    private static function shouldApplyLateFee($dueDate, $paymentStatus, $terms) {
        // Skip if no due date
        if (empty($dueDate)) {
            return false;
        }

        // Skip if already paid
        if (in_array($paymentStatus, ['Paid', 'Refunded', 'Partially Refunded'])) {
            return false;
        }

        // Skip if Net-0 (immediate payment)
        if ($terms === 'Net-0' || $terms === 'Net-0 (Immediate)') {
            return false;
        }

        // Check if late fees are enabled globally
        $enabled = \COption::GetOptionString('late_fees', 'enabled', 'Y') === 'Y';
        if (!$enabled) {
            return false;
        }

        return true;
    }

    /**
     * Get state code from Deal
     */
    private static function getStateFromDeal($deal) {
        // From custom field: UF_CRM_SERVICE_AREA
        $stateCode = $deal['UF_CRM_SERVICE_AREA'] ?? null;

        if (!$stateCode) {
            // Try to extract from contact address
            $contactId = $deal['CONTACT_ID'];
            if ($contactId) {
                $contact = \CCrmContact::GetByID($contactId, false);
                if ($contact && isset($contact['ADDRESS_REGION'])) {
                    $stateCode = $contact['ADDRESS_REGION'];
                }
            }
        }

        return $stateCode ?? 'TX'; // Default to Texas
    }

    /**
     * Get late fee configuration for state
     */
    private static function getLateFeeConfig($stateCode) {
        $settings = json_decode(\COption::GetOptionString('late_fees', 'states', '[]'), true);

        // Return state-specific config if exists and enabled
        if (isset($settings[$stateCode]) && ($settings[$stateCode]['enabled'] ?? false)) {
            return $settings[$stateCode];
        }

        // Return default configuration
        $defaultConfig = json_decode(\COption::GetOptionString('late_fees', 'default', '{}'), true);
        return [
            'rate' => $defaultConfig['rate'] ?? 1.5,
            'grace_period' => $defaultConfig['grace_period'] ?? 5,
            'max_fee' => $defaultConfig['max_fee'] ?? null
        ];
    }

    /**
     * Get invoice amount (excluding late fees)
     */
    private static function getInvoiceAmount($dealId) {
        global $DB;

        $result = $DB->Query("
            SELECT SUM(PRICE * QUANTITY) as amount
            FROM b_sale_basket
            WHERE ORDER_ID = (
                SELECT ID FROM b_sale_order WHERE ACCOUNT_NUMBER = '{$dealId}'
            )
            AND NAME NOT LIKE '%Late Payment Fee%'
        ")->Fetch();

        return floatval($result['amount'] ?? 0);
    }

    /**
     * Apply late fee to invoice
     * Adds or updates late fee line item
     */
    public static function applyLateFee($dealId) {
        // Calculate fee
        $feeData = self::calculateLateFee($dealId);

        if ($feeData['fee'] <= 0) {
            return; // No fee to apply
        }

        // Get or create order
        $order = \Bitrix\Sale\Order::loadByAccountNumber($dealId);
        if (!$order) {
            throw new \Exception('Order not found');
        }

        $basket = $order->getBasket();
        $lateFeeItemFound = false;

        // Check if late fee already exists
        foreach ($basket as $item) {
            if (strpos($item->getField('NAME'), 'Late Payment Fee') !== false) {
                // Update existing late fee
                $item->setField('PRICE', $feeData['fee']);
                $item->setField('NAME', sprintf(
                    'Late Payment Fee (%d days overdue)',
                    $feeData['days_overdue']
                ));
                $item->save();
                $lateFeeItemFound = true;
                break;
            }
        }

        // Add new late fee item if not found
        if (!$lateFeeItemFound) {
            $item = $basket->createItem('catalog', 0);
            $item->setFields([
                'NAME' => sprintf('Late Payment Fee (%d days overdue)', $feeData['days_overdue']),
                'PRICE' => $feeData['fee'],
                'QUANTITY' => 1,
                'CUSTOM_PRICE' => 'Y',
                'CURRENCY' => 'USD',
                'LID' => SITE_ID
            ]);
        }

        $order->save();

        // Update Deal custom field
        \CCrmDeal::Update($dealId, [
            'UF_CRM_LATE_FEE' => $feeData['fee']
        ]);

        // Add timeline entry
        $comment = sprintf(
            "⏰ Late payment fee applied: $%s\n" .
            "Rate: %s%% per month\n" .
            "Days overdue: %d\n" .
            "Grace period: %d days",
            number_format($feeData['fee'], 2),
            $feeData['rate'],
            $feeData['days_overdue'],
            $feeData['grace_period']
        );

        if ($feeData['capped']) {
            $comment .= "\n(Capped at maximum: $" . number_format($feeData['max_fee'], 2) . ")";
        }

        \Bitrix\Crm\Timeline\CommentController::getInstance()->onCreate([
            'ENTITY_TYPE_ID' => \CCrmOwnerType::Deal,
            'ENTITY_ID' => $dealId,
            'COMMENT' => $comment
        ]);

        // Notify manager if fee exceeds threshold
        $threshold = \COption::GetOptionFloat('late_fees', 'notify_threshold', 50.00);
        if ($feeData['fee'] >= $threshold) {
            self::notifyManagerAboutLateFee($dealId, $feeData);
        }
    }

    /**
     * Remove late fee from invoice
     */
    public static function removeLateFee($dealId) {
        $order = \Bitrix\Sale\Order::loadByAccountNumber($dealId);
        if (!$order) {
            return;
        }

        $basket = $order->getBasket();
        foreach ($basket as $item) {
            if (strpos($item->getField('NAME'), 'Late Payment Fee') !== false) {
                $item->delete();
                break;
            }
        }

        $order->save();

        // Clear Deal field
        \CCrmDeal::Update($dealId, [
            'UF_CRM_LATE_FEE' => 0
        ]);
    }

    /**
     * Notify manager about high-value late fee
     */
    private static function notifyManagerAboutLateFee($dealId, $feeData) {
        $deal = \CCrmDeal::GetByID($dealId);
        if (!$deal) {
            return;
        }

        $managerId = $deal['ASSIGNED_BY_ID'] ?? 1;

        \CIMNotify::Add([
            'TO_USER_ID' => $managerId,
            'NOTIFY_TYPE' => \IM_NOTIFY_SYSTEM,
            'NOTIFY_MODULE' => 'crm',
            'NOTIFY_MESSAGE' => "⏰ Late payment fee applied to Deal #{$dealId}\n" .
                              "Fee: $" . number_format($feeData['fee'], 2) . "\n" .
                              "Days overdue: {$feeData['days_overdue']}\n" .
                              "[View Deal](/crm/deal/details/{$dealId}/)"
        ]);
    }
}
```

## 4. CRON Job Implementation

### 4.1. Daily Late Fee Check

**File:** `/local/php_interface/cron/check_late_payment_fees.php`

```php
<?php
/**
 * Daily Late Payment Fee Check
 *
 * Schedule: 0 2 * * * (2:00 AM daily)
 *
 * Usage: php /local/php_interface/cron/check_late_payment_fees.php
 */

require_once($_SERVER['DOCUMENT_ROOT'] . '/bitrix/modules/main/include/prolog_before.php');

use Local\Payment\LateFeeCalculator;

// Check if CRON is enabled
$cronEnabled = \COption::GetOptionString('late_fees', 'cron_enabled', 'Y') === 'Y';
if (!$cronEnabled) {
    echo "CRON job disabled in settings\n";
    exit(0);
}

echo "Starting late payment fee check...\n";
echo "Time: " . date('Y-m-d H:i:s') . "\n\n";

// Get all unpaid deals with due dates in the past
$arFilter = [
    '!UF_CRM_PAYMENT_STATUS' => ['Paid', 'Refunded', 'Partially Refunded'],
    '!UF_CRM_TERMS_OF_PAYMENT' => ['Net-0', 'Net-0 (Immediate)'],
    '<=UF_CRM_INVOICE_DUE_DATE' => date('d.m.Y'),
    '>UF_CRM_INVOICE_DUE_DATE' => '01.01.2020' // Has a due date set
];

$dealsResult = \CCrmDeal::GetList(
    ['ID' => 'ASC'],
    $arFilter,
    false,
    false,
    ['ID', 'TITLE', 'UF_CRM_INVOICE_DUE_DATE', 'UF_CRM_LATE_FEE', 'UF_CRM_SERVICE_AREA']
);

$processedCount = 0;
$skippedCount = 0;
$errorCount = 0;

while ($deal = $dealsResult->Fetch()) {
    try {
        $dealId = $deal['ID'];
        $currentFee = floatval($deal['UF_CRM_LATE_FEE'] ?? 0);

        // Calculate what the fee should be
        $feeData = LateFeeCalculator::calculateLateFee($dealId);

        // Only update if fee amount changed
        if (abs($feeData['fee'] - $currentFee) >= 0.01) {
            LateFeeCalculator::applyLateFee($dealId);

            echo "✓ Deal #{$dealId}: Late fee updated to $" . number_format($feeData['fee'], 2) .
                 " ({$feeData['days_overdue']} days overdue)\n";
            $processedCount++;
        } else {
            echo "- Deal #{$dealId}: No change (current: $" . number_format($currentFee, 2) . ")\n";
            $skippedCount++;
        }

    } catch (\Exception $e) {
        echo "✗ Deal #{$dealId}: Error - " . $e->getMessage() . "\n";
        $errorCount++;
    }
}

echo "\n";
echo "Summary:\n";
echo "  Processed: {$processedCount}\n";
echo "  Skipped: {$skippedCount}\n";
echo "  Errors: {$errorCount}\n";
echo "  Total: " . ($processedCount + $skippedCount + $errorCount) . "\n";

echo "\nCompleted at: " . date('Y-m-d H:i:s') . "\n";
```
