# TR-15: Payment Integration - Sunbit Financing

**Epic:** Payment Processing System
**Component:** Sunbit Installment Financing Integration
**Version:** 1.0
**Last Updated:** January 16, 2025

---

## 1. Overview

### 1.1. Purpose

Integrate Sunbit installment financing as a payment method for clients who need to finance their invoices over 6-24 months. This provides:
- Flexible payment terms (6, 12, 18, 24 months)
- Higher approval rates for clients
- Immediate payment to company (Sunbit handles all collections)
- Simple redirect integration to Sunbit platform

### 1.2. Business Requirements

**When to Offer Financing:**
- Available for ANY invoice amount (no minimum)
- All job types (residential, commercial, automotive)
- Available in Client Portal as payment method option
- Client decides if they want financing

**Benefits:**
- **For Client:** Affordable monthly payments, instant approval, soft credit check only
- **For Company:** Get paid immediately by Sunbit (95% of invoice), zero credit risk, no collections
- **For Technician:** Close jobs on-site regardless of client's immediate cash availability

### 1.3. Sunbit Fee Structure

**Fixed Rate:**
- **Merchant Discount Rate:** 5% (flat rate for all amounts)
- **Company Receives:** 95% of invoice total
- **Disbursement:** Next business day after approval

**Example:**
```
Invoice Total: $1,850.00
Sunbit Fee (5%): $92.50
Company Receives: $1,757.50 (disbursed next business day)

Client's Loan Terms (12 months @ 9.99% APR):
  Monthly Payment: $162.50
  Total Paid by Client: $1,950.00

Note: Client pays APR to Sunbit (not included in invoice total)
```

**How it Works:**
1. Client selects "Sunbit Financing" as payment method
2. System redirects to Sunbit platform
3. Sunbit sends application link to client
4. Client fills out application (SSN, income, etc.)
5. Sunbit checks credit score and approves/declines
6. If approved: Credit line opens, company gets paid 95%
7. Client makes monthly payments directly to Sunbit

#### 3.1.2. Sunbit API Client

File: `/local/php_interface/classes/SunbitClient.php`

```php
<?php
namespace Local\Payment;

use Bitrix\Main\Web\HttpClient;
use Bitrix\Main\Web\Json;

class SunbitClient {
    private $merchantId;
    private $apiKey;
    private $apiEndpoint;
    private $httpClient;

    public function __construct() {
        $config = \COption::GetOptionString('sunbit', 'config', '[]');
        $config = Json::decode($config);

        $this->merchantId = $config['merchant_id'];
        $this->apiKey = $config['api_key'];
        $this->apiEndpoint = $config['api_endpoint'];

        $this->httpClient = new HttpClient([
            'headers' => [
                'Content-Type: application/json',
                'Authorization: Bearer ' . $this->apiKey,
                'X-Merchant-ID: ' . $this->merchantId
            ]
        ]);
    }

    /**
     * Create financing application
     *
     * @param array $params Application parameters
     * @return array Response with application_id and application_url
     */
    public function createApplication($params) {
        $endpoint = $this->apiEndpoint . '/applications';

        $payload = [
            'merchant_id' => $this->merchantId,
            'amount' => floatval($params['amount']),
            'customer' => [
                'first_name' => $params['customer']['first_name'],
                'last_name' => $params['customer']['last_name'],
                'email' => $params['customer']['email'],
                'phone' => $params['customer']['phone'],
                'address' => [
                    'street' => $params['customer']['address']['street'] ?? '',
                    'city' => $params['customer']['address']['city'] ?? '',
                    'state' => $params['customer']['address']['state'] ?? '',
                    'zip' => $params['customer']['address']['zip'] ?? ''
                ]
            ],
            'metadata' => [
                'bitrix_deal_id' => $params['deal_id'],
                'invoice_number' => $params['invoice_number'],
                'created_by' => $params['created_by'] ?? 'system'
            ],
            'return_url' => $params['return_url'],
            'cancel_url' => $params['cancel_url']
        ];

        $response = $this->httpClient->post($endpoint, Json::encode($payload));

        if ($this->httpClient->getStatus() !== 200) {
            throw new \Exception('Sunbit API error: ' . $response);
        }

        $data = Json::decode($response);

        // Log application creation
        $this->logApplication($params['deal_id'], $data['application_id'], 'created');

        return [
            'application_id' => $data['application_id'],
            'application_url' => $data['application_url'],
            'expires_at' => $data['expires_at']
        ];
    }

    /**
     * Get application status
     *
     * @param string $applicationId Sunbit application ID
     * @return array Application details
     */
    public function getApplication($applicationId) {
        $endpoint = $this->apiEndpoint . '/applications/' . $applicationId;

        $response = $this->httpClient->get($endpoint);

        if ($this->httpClient->getStatus() !== 200) {
            throw new \Exception('Sunbit API error: ' . $response);
        }

        return Json::decode($response);
    }

    /**
     * Get loan details (after approval)
     *
     * @param string $loanId Sunbit loan ID
     * @return array Loan details
     */
    public function getLoan($loanId) {
        $endpoint = $this->apiEndpoint . '/loans/' . $loanId;

        $response = $this->httpClient->get($endpoint);

        if ($this->httpClient->getStatus() !== 200) {
            throw new \Exception('Sunbit API error: ' . $response);
        }

        return Json::decode($response);
    }

    /**
     * Log Sunbit transaction
     */
    private function logApplication($dealId, $applicationId, $status) {
        global $DB;

        $DB->Query("
            INSERT INTO b_sunbit_applications
            (DEAL_ID, APPLICATION_ID, STATUS, CREATED_DATE)
            VALUES (
                " . intval($dealId) . ",
                '" . $DB->ForSql($applicationId) . "',
                '" . $DB->ForSql($status) . "',
                NOW()
            )
        ");
    }
}
```

#### 3.2.2. AJAX Handler for Application Creation

File: `/local/php_interface/classes/Controllers/SunbitController.php`

```php
<?php
namespace Local\Payment\Controllers;

use Bitrix\Main\Engine\Controller;
use Bitrix\Main\Engine\ActionFilter;
use Local\Payment\SunbitClient;

class SunbitController extends Controller {

    public function configureActions() {
        return [
            'createApplication' => [
                'prefilters' => [
                    new ActionFilter\Authentication(),
                    new ActionFilter\Csrf()
                ]
            ]
        ];
    }

    /**
     * Create Sunbit financing application
     *
     * @param int $dealId Deal ID
     * @param float $amount Invoice total
     * @return array
     */
    public function createApplicationAction($dealId, $amount) {
        try {
            // Get Deal details
            $deal = \CCrmDeal::GetByID($dealId);
            if (!$deal) {
                throw new \Exception('Deal not found');
            }

            // Get client contact details
            $contactId = $deal['CONTACT_ID'];
            $contact = \CCrmContact::GetByID($contactId);

            // Get client address from Deal
            $address = $this->getClientAddress($dealId);

            // Initialize Sunbit client
            $sunbit = new SunbitClient();

            // Create application
            $result = $sunbit->createApplication([
                'deal_id' => $dealId,
                'invoice_number' => $dealId, // Invoice # = Deal ID
                'amount' => $amount,
                'customer' => [
                    'first_name' => $contact['NAME'],
                    'last_name' => $contact['LAST_NAME'],
                    'email' => $contact['EMAIL'][0]['VALUE'] ?? '',
                    'phone' => $contact['PHONE'][0]['VALUE'] ?? '',
                    'address' => $address
                ],
                'return_url' => "https://{$_SERVER['HTTP_HOST']}/client-portal/invoice/{$dealId}/payment-complete",
```

```php
<?php
require($_SERVER['DOCUMENT_ROOT'] . '/bitrix/modules/main/include/prolog_before.php');

use Bitrix\Main\Web\Json;
use Local\Payment\SunbitClient;

/**
 * Sunbit Webhook Endpoint
 * URL: https://bitrix24.yourdomain.com/local/php_interface/webhooks/sunbit_webhook.php
 */

// Get raw POST data
$payload = @file_get_contents('php://input');
$signature = $_SERVER['HTTP_X_SUNBIT_SIGNATURE'] ?? '';

// Verify webhook signature
$webhookSecret = \COption::GetOptionString('sunbit', 'webhook_secret');
$expectedSignature = hash_hmac('sha256', $payload, $webhookSecret);

if (!hash_equals($expectedSignature, $signature)) {
    http_response_code(401);
    echo Json::encode(['error' => 'Invalid signature']);
    exit;
}

// Parse event
$event = Json::decode($payload);

// Log webhook event
logSunbitWebhook($event);

// Process event based on type
switch ($event['type']) {
    case 'application.submitted':
        handleApplicationSubmitted($event['data']);
        break;

    case 'application.approved':
        handleApplicationApproved($event['data']);
        break;

    case 'application.declined':
        handleApplicationDeclined($event['data']);
        break;

    case 'loan.funded':
        handleLoanFunded($event['data']);
        break;

    case 'loan.paid_off':
        handleLoanPaidOff($event['data']);
        break;

    case 'loan.defaulted':
        handleLoanDefaulted($event['data']);
        break;

    default:
        // Unknown event type
        break;
}

http_response_code(200);
echo Json::encode(['success' => true]);

    $payment->setField('PAID', 'Y');

        'ENTITY_TYPE_ID' => \CCrmOwnerType::Deal,
        'ENTITY_ID' => $dealId,
```

### 8.3. Comparison: Sunbit vs Stripe Payment Plans

| Feature | Sunbit | Stripe Payment Plans |

| **Company Risk** | Zero (Sunbit pays 95% upfront) | High (company extends credit) |

| **Collections** | Sunbit handles 100% | Company handles |
| **Default Risk** | Sunbit assumes all risk | Company assumes all risk |
| **Terms** | 6-24 months (client chooses) | Flexible |
| **Minimum Amount** | None (available for any amount) | No minimum |
| **Client APR** | Client pays to Sunbit | Included in invoice |
