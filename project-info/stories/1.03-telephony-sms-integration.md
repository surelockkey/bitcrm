# TR-03: SMS Integration

**Epic:** Telephony Integration
**Document Version:** 1.0
**Date:** 2025-10-10
**Status:** Draft

## 1. Overview

### 1.1 Purpose
Ensure two-way SMS communication between dispatchers and clients through Bitrix24 with full integration into CRM timeline, using Twilio Programmable SMS.

### 1.2 Scope
- SMS from Bitrix24 CRM (Deal/Contact/Lead) to client (dispatchers only)
- Inbound SMS from clients to dispatchers
- Automatic binding of SMS to corresponding CRM object
- Storage of SMS history in Deal/Contact timeline
- SMS templates for common messages

## 2. Architecture

### 2.2 Numbers for SMS

**Dispatcher → Client:**
- Uses `UF_CRM_CALLED_NUMBER` from Deal (the number the client called)
- If Deal doesn't exist or field is empty → uses service area number based on Contact address
- This ensures the client sees a familiar company number

**Client → Dispatcher:**
- All 12 service area numbers configured in Twilio to receive SMS
- Inbound SMS routed to Bitrix24 Open Channels
- System automatically searches for active Deal or Contact by client number

#### 3.1.2 Caller ID Selection Logic
```php
function getSmsFromNumber($dealId = null, $contactId = null) {
    // Priority 1: number the client called (from Deal)
    if ($dealId) {
        $deal = \CCrmDeal::GetByID($dealId);
        $calledNumber = $deal['UF_CRM_CALLED_NUMBER'];

        if ($calledNumber && isValidTwilioNumber($calledNumber)) {
            return $calledNumber;
        }

        // If Deal has service area number
        $serviceAreaNumber = $deal['UF_CRM_SERVICE_AREA_NUMBER'];
        if ($serviceAreaNumber && isValidTwilioNumber($serviceAreaNumber)) {
            return $serviceAreaNumber;
        }
    }

    // Priority 2: service area number based on Contact address
    if ($contactId) {
        $contact = \CCrmContact::GetByID($contactId);
        $state = extractStateFromAddress($contact);
        $clientType = $contact['UF_CRM_CLIENT_TYPE'] ?? 'Regular';

        $mapping = ServiceAreaNumberMapping::getNumberForState($state, $clientType);
        if ($mapping) {
            return $mapping;
        }
    }

    // Fallback: default Texas Regular number
    return '+14695521001';
}
```

#### 3.1.3 REST API Implementation
```php
// /rest/sms/send
class SmsController {
    public function sendAction($dealId, $contactId, $toNumber, $message) {
        $fromNumber = getSmsFromNumber($dealId, $contactId);

        // Send via Twilio
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        $params = [
            'from' => $fromNumber,
            'body' => $message
        ];

        try {
            $twilioMessage = $twilioClient->messages->create($toNumber, $params);

            // Save to Bitrix timeline
            $this->saveToTimeline([
                'ENTITY_TYPE' => $dealId ? 'DEAL' : 'CONTACT',
                'ENTITY_ID' => $dealId ?: $contactId,
                'TYPE' => 'SMS_OUTBOUND',
                'FROM' => $fromNumber,
                'TO' => $toNumber,
                'MESSAGE' => $message,
                'TWILIO_SID' => $twilioMessage->sid,
                'STATUS' => $twilioMessage->status,
                'DIRECTION' => 'OUTBOUND',
                'TIMESTAMP' => new DateTime()
            ]);

            return [
                'success' => true,
                'messageSid' => $twilioMessage->sid,
                'status' => $twilioMessage->status
            ];

        } catch (\Exception $e) {
            // Log error and save failed message
            $this->saveToTimeline([
                'ENTITY_TYPE' => $dealId ? 'DEAL' : 'CONTACT',
                'ENTITY_ID' => $dealId ?: $contactId,
                'TYPE' => 'SMS_OUTBOUND',
                'FROM' => $fromNumber,
                'TO' => $toNumber,
                'MESSAGE' => $message,
                'STATUS' => 'failed',
                'ERROR' => $e->getMessage(),
                'DIRECTION' => 'OUTBOUND',
                'TIMESTAMP' => new DateTime()
            ]);

            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }
}
```

### 3.2 Inbound SMS (Client → Dispatcher)

#### 3.2.1 Twilio Webhook Handler
```php
// /rest/twilio/sms-inbound
class TwilioSmsWebhook {
    public function inboundAction(Request $request) {
        // Validate Twilio signature
        $validator = new RequestValidator(TWILIO_AUTH_TOKEN);
        $signature = $request->headers->get('X-Twilio-Signature');

        if (!$validator->validate($signature, $request->getUri(), $request->request->all())) {
            return new Response('Unauthorized', 403);
        }

        $data = [
            'MessageSid' => $request->get('MessageSid'),
            'From' => $request->get('From'),           // Client phone
            'To' => $request->get('To'),               // Company number
            'Body' => $request->get('Body')
        ];

        // Find CRM entity (Deal or Contact)
        $entity = $this->findCrmEntity($data['From']);

        if ($entity) {
            // Save to timeline
            $this->saveInboundSms($entity, $data);

            // Notify dispatchers via Open Channels
            $this->notifyDispatchers($entity, $data);
        } else {
            // Create new Lead
            $leadId = $this->createLeadFromSms($data);
            $this->notifyDispatchers(['type' => 'LEAD', 'id' => $leadId], $data);
        }

        return new Response('<?xml version="1.0" encoding="UTF-8"?><Response></Response>', 200, [
            'Content-Type' => 'text/xml'
        ]);
    }

    private function findCrmEntity($phone) {
        $phone = normalizePhone($phone);

        // 1. Search for active Deal with this phone
        $deals = \CCrmDeal::GetListEx(
            ['DATE_CREATE' => 'DESC'],
            [
                'PHONE' => $phone,
                'STAGE_SEMANTIC_ID' => ['P', 'S'],  // In progress or success
                '!CLOSED' => 'Y'
            ],
            false,
            ['nTopCount' => 1]
        );

        if ($deal = $deals->Fetch()) {
            return ['type' => 'DEAL', 'id' => $deal['ID']];
        }

        // 2. Search for Contact
        $contacts = \CCrmContact::GetListEx(
            ['DATE_MODIFY' => 'DESC'],
            ['PHONE' => $phone],
            false,
            ['nTopCount' => 1]
        );

        if ($contact = $contacts->Fetch()) {
            return ['type' => 'CONTACT', 'id' => $contact['ID']];
        }

        // 3. Not found
        return null;
    }

    private function saveInboundSms($entity, $data) {
        $timelineEntry = [
            'ENTITY_TYPE' => $entity['type'],
            'ENTITY_ID' => $entity['id'],
            'TYPE' => 'SMS_INBOUND',
            'FROM' => $data['From'],
            'TO' => $data['To'],
            'MESSAGE' => $data['Body'],
            'TWILIO_SID' => $data['MessageSid'],
            'STATUS' => 'received',
            'DIRECTION' => 'INBOUND',
            'TIMESTAMP' => new DateTime()
        ];

        // Save to custom table for SMS history
        global $DB;
        $DB->Query("
            INSERT INTO b_crm_sms_log (
                ENTITY_TYPE, ENTITY_ID, DIRECTION,
                FROM_NUMBER, TO_NUMBER, MESSAGE_TEXT,
                TWILIO_SID, STATUS, DATE_CREATE
            ) VALUES (
                '{$timelineEntry['ENTITY_TYPE']}',
                {$timelineEntry['ENTITY_ID']},
                '{$timelineEntry['DIRECTION']}',
                '{$timelineEntry['FROM']}',
                '{$timelineEntry['TO']}',
                '{$DB->ForSql($timelineEntry['MESSAGE'])}',
                '{$timelineEntry['TWILIO_SID']}',
                '{$timelineEntry['STATUS']}',
                NOW()
            )
        ");

        // Add to CRM timeline
        \Bitrix\Crm\Timeline\ActivityController::getInstance()->onCreate(
            $entity['id'],
            [
                'TYPE_ID' => \CCrmActivityType::Provider,
                'PROVIDER_ID' => 'SMS_PROVIDER',
                'SUBJECT' => 'SMS from ' . formatPhone($data['From']),
                'DESCRIPTION' => $data['Body'],
                'COMMUNICATIONS' => [[
                    'ENTITY_TYPE_ID' => \CCrmOwnerType::Contact,
                    'VALUE' => $data['From'],
                    'ENTITY_ID' => $entity['type'] === 'DEAL' ? getDealContactId($entity['id']) : $entity['id']
                ]]
            ]
        );
    }

    private function notifyDispatchers($entity, $data) {
        // Get all online dispatchers
        $dispatchers = \Bitrix\Im\User::getInstance()->getActiveUsers([
            'FILTER' => [
                'UF_DEPARTMENT' => DISPATCHER_DEPARTMENT_ID
            ]
        ]);

        $messageText = "📱 New SMS from " . formatPhone($data['From']) . "\n";
        $messageText .= "Deal: " . ($entity['type'] === 'DEAL' ? "#" . $entity['id'] : "none") . "\n";
        $messageText .= "Text: " . $data['Body'];

        foreach ($dispatchers as $dispatcher) {
            \CIMNotify::Add([
                'TO_USER_ID' => $dispatcher['ID'],
                'FROM_USER_ID' => 0,  // System notification
                'NOTIFY_TYPE' => \IM_NOTIFY_SYSTEM,
                'NOTIFY_MODULE' => 'crm',
                'NOTIFY_EVENT' => 'sms_inbound',
                'NOTIFY_TAG' => 'SMS|' . $entity['type'] . '|' . $entity['id'],
                'NOTIFY_MESSAGE' => $messageText,
                'NOTIFY_MESSAGE_OUT' => $messageText
            ]);
        }
    }

    private function createLeadFromSms($data) {
        $lead = new \CCrmLead(false);

        $fields = [
            'TITLE' => 'SMS Lead: ' . formatPhone($data['From']),
            'NAME' => '',
            'LAST_NAME' => '',
            'PHONE' => [['VALUE' => $data['From'], 'VALUE_TYPE' => 'MOBILE']],
            'SOURCE_ID' => 'SMS',
            'SOURCE_DESCRIPTION' => 'Inbound SMS to ' . formatPhone($data['To']),
            'COMMENTS' => $data['Body'],
            'ASSIGNED_BY_ID' => DISPATCHER_LEAD_ROUTER_USER_ID,
            'OPENED' => 'Y'
        ];

        $leadId = $lead->Add($fields);

        // Save SMS to timeline
        $this->saveInboundSms(['type' => 'LEAD', 'id' => $leadId], $data);

        return $leadId;
    }
}
```

#### 3.3.2 Placeholder Replacement
```php
class SmsTemplateEngine {
    public static function renderTemplate($templateId, $dealId = null, $customData = []) {
        global $DB;

        // Get template
        $template = $DB->Query("
            SELECT MESSAGE FROM b_crm_sms_template
            WHERE ID = {$templateId}
        ")->Fetch();

        if (!$template) {
            throw new Exception("Template not found");
        }

        $message = $template['MESSAGE'];

        // Get Deal data if provided
        if ($dealId) {
            $deal = \CCrmDeal::GetByID($dealId);
            $contact = \CCrmContact::GetByID($deal['CONTACT_ID']);

            $placeholders = [
                '{CLIENT_NAME}' => $contact['NAME'] . ' ' . $contact['LAST_NAME'],
                '{COMPANY_NUMBER}' => $deal['UF_CRM_CALLED_NUMBER'] ?? $deal['UF_CRM_SERVICE_AREA_NUMBER'],
                '{APPOINTMENT_DATE}' => date('m/d/Y', strtotime($deal['UF_CRM_APPOINTMENT_DATE'])),
                '{APPOINTMENT_TIME}' => date('g:i A', strtotime($deal['UF_CRM_APPOINTMENT_TIME'])),
                '{AMOUNT}' => number_format($deal['OPPORTUNITY'], 2),
                '{INVOICE_NUMBER}' => $deal['UF_CRM_INVOICE_NUMBER']
            ];

            // Merge custom data
            $placeholders = array_merge($placeholders, $customData);

            $message = str_replace(
                array_keys($placeholders),
                array_values($placeholders),
                $message
            );
        }

        return $message;
    }
}
```

## 6. Error Handling

### 6.1 Twilio Errors
| Error Code | Reason | System Action |
|------------|---------|-------------|
| 21211 | Invalid 'To' phone number | Show error to dispatcher: "Invalid phone number format" |
| 21408 | Permission to send SMS denied | Log, notify administrator, show error |
| 21610 | Message cannot be sent (landline) | Show: "This number does not support SMS" |
| 30003 | Unreachable destination | Save as "failed", retry in 5 minutes |
| 30005 | Unknown destination | Show: "Number unavailable" |

### 6.2 Fallback Strategy
```php
class SmsFailureHandler {
    public function handleFailedSms($smsLogId) {
        global $DB;

        $sms = $DB->Query("
            SELECT * FROM b_crm_sms_log
            WHERE ID = {$smsLogId}
        ")->Fetch();

        // Retry logic for temporary failures (30003)
        if ($sms['ERROR_CODE'] == '30003' && $sms['RETRY_COUNT'] < 3) {
            sleep(300);  // Wait 5 minutes
            $this->retrySms($smsLogId);
            return;
        }

        // For permanent failures, notify dispatcher
        $this->notifyDispatcherOfFailure($sms);

        // Create manual task to call client instead
        $this->createFollowupTask($sms);
    }

    private function createFollowupTask($sms) {
        \CTaskItem::add([
            'TITLE' => 'Failed SMS - call client instead',
            'DESCRIPTION' => "SMS to {$sms['TO_NUMBER']} failed.\nOriginal message: {$sms['MESSAGE_TEXT']}",
            'RESPONSIBLE_ID' => getCurrentDispatcher(),
            'DEADLINE' => date('Y-m-d H:i:s', strtotime('+30 minutes')),
            'UF_CRM_TASK' => ["D_{$sms['ENTITY_ID']}"]
        ], getCurrentDispatcher());
    }
}
```

```php
// Middleware for all Twilio endpoints
class TwilioWebhookMiddleware {
    public function handle(Request $request, Closure $next) {
        $validator = new \Twilio\Security\RequestValidator(TWILIO_AUTH_TOKEN);

        $signature = $request->headers->get('X-Twilio-Signature');
        $url = $request->getUri();
        $params = $request->request->all();

        if (!$validator->validate($signature, $url, $params)) {
            \Bitrix\Main\Diag\Logger::log(
                'Twilio webhook authentication failed',
                'SECURITY',
                [
                    'url' => $url,
                    'ip' => $request->getClientIp(),
                    'signature' => $signature
                ]
            );

            return new Response('Forbidden', 403);
        }

        return $next($request);
    }
}
```

## 8. Analytics & Reporting

### 8.1 SMS Metrics Dashboard
```sql
-- SMS volume by direction
SELECT
    DATE(DATE_CREATE) as date,
    DIRECTION,
    COUNT(*) as count,
    SUM(CASE WHEN STATUS = 'delivered' THEN 1 ELSE 0 END) as delivered,
    SUM(CASE WHEN STATUS = 'failed' THEN 1 ELSE 0 END) as failed
FROM b_crm_sms_log
WHERE DATE_CREATE >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY DATE(DATE_CREATE), DIRECTION
ORDER BY date DESC;

-- Top SMS templates usage
SELECT
    t.TITLE,
    COUNT(l.ID) as usage_count,
    AVG(CASE WHEN l.STATUS = 'delivered' THEN 1 ELSE 0 END) * 100 as delivery_rate
FROM b_crm_sms_template t
LEFT JOIN b_crm_sms_log l ON l.MESSAGE_TEXT LIKE CONCAT('%', SUBSTRING(t.MESSAGE, 1, 50), '%')
WHERE t.IS_ACTIVE = 'Y'
GROUP BY t.ID
ORDER BY usage_count DESC;

-- Response time (from client SMS to dispatcher reply)
SELECT
    AVG(TIMESTAMPDIFF(SECOND, inbound.DATE_CREATE, outbound.DATE_CREATE)) / 60 as avg_response_minutes
FROM b_crm_sms_log inbound
JOIN b_crm_sms_log outbound
    ON inbound.ENTITY_TYPE = outbound.ENTITY_TYPE
    AND inbound.ENTITY_ID = outbound.ENTITY_ID
    AND outbound.DATE_CREATE > inbound.DATE_CREATE
    AND outbound.DATE_CREATE < DATE_ADD(inbound.DATE_CREATE, INTERVAL 1 HOUR)
WHERE inbound.DIRECTION = 'INBOUND'
  AND outbound.DIRECTION = 'OUTBOUND'
  AND inbound.DATE_CREATE >= DATE_SUB(NOW(), INTERVAL 7 DAY);
```

### 9.2 Message Retention
```php
// Cron job: cleanup old SMS logs (retain 2 years)
class SmsLogCleanup {
    public function execute() {
        global $DB;

        $cutoffDate = date('Y-m-d', strtotime('-2 years'));

        // Archive to backup table before deletion
        $DB->Query("
            INSERT INTO b_crm_sms_log_archive
            SELECT * FROM b_crm_sms_log
            WHERE DATE_CREATE < '{$cutoffDate}'
        ");

        // Delete from main table
        $DB->Query("
            DELETE FROM b_crm_sms_log
            WHERE DATE_CREATE < '{$cutoffDate}'
        ");

        \Bitrix\Main\Diag\Logger::log(
            "SMS log cleanup completed",
            "SMS_CLEANUP"
        );
    }
}
```

### 9.3 TCPA Compliance
- Opt-out mechanism: client sends "STOP" → automatically added to blacklist

```php
class TcpaCompliance {
    public function canSendSms($contactId, $dealId) {
        // Check opt-out status
        if ($this->isOptedOut($contactId)) {
            return ['allowed' => false, 'reason' => 'Client opted out'];
        }

        return ['allowed' => true];
    }

    private function handleStopMessage($contactId, $phone) {
        // Add to opt-out list
        global $DB;
        $DB->Query("
            INSERT INTO b_crm_sms_optout (CONTACT_ID, PHONE, DATE_OPTOUT)
            VALUES ({$contactId}, '{$phone}', NOW())
            ON DUPLICATE KEY UPDATE DATE_OPTOUT = NOW()
        ");

        // Send confirmation
        $this->sendConfirmationSms($phone, "You have been unsubscribed from SMS notifications. Reply START to re-subscribe.");
    }
}
```
