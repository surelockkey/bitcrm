## 3. Call Control Functions

### 3.1 Call Transfer

#### 3.1.1 Transfer Type

**Warm Transfer (Consultation Transfer):**
```
Dispatcher A talks with Client
    ↓
Dispatcher A clicks "Transfer" → selects Dispatcher B
    ↓
Client on hold (music)
    ↓
System calls Dispatcher B
    ↓
Dispatcher B answers → Dispatcher A briefs about client
    ↓
Dispatcher A clicks "Complete Transfer"
    ↓
Client connects with Dispatcher B
Dispatcher A disconnected
```

#### 3.1.2 Transfer Implementation

```php
// /local/api/telephony/transfer.php

class CallTransferController {
    /**
     * Initiates warm transfer (consultation)
     */
    public function warmTransferAction($callSid, $targetUserId) {
        global $USER;

        // Get target user phone
        $targetUser = \CUser::GetByID($targetUserId)->Fetch();
        $targetPhone = $targetUser['PERSONAL_PHONE'];

        if (!$targetPhone) {
            return [
                'success' => false,
                'error' => 'Target user has no phone configured'
            ];
        }

        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        try {
            // Step 1: Put current call on hold
            $call = $twilioClient->calls($callSid)->update([
                'twiml' => '<Response><Say>Please hold while we transfer your call.</Say><Play loop="100">' . HOLD_MUSIC_URL . '</Play></Response>'
            ]);

            // Step 2: Create consultation call to target user
            $consultCall = $twilioClient->calls->create(
                $targetPhone, // To
                COMPANY_PHONE, // From
                [
                    'url' => SITE_URL . '/local/webhooks/twilio/transfer_consult.php?originalCallSid=' . $callSid,
                    'statusCallback' => SITE_URL . '/local/webhooks/twilio/transfer_status.php',
                    'statusCallbackEvent' => ['answered', 'completed']
                ]
            );

            // Store transfer state
            $this->saveTransferState([
                'original_call_sid' => $callSid,
                'consult_call_sid' => $consultCall->sid,
                'target_user_id' => $targetUserId,
                'initiator_user_id' => $USER->GetID(),
                'status' => 'CONSULTING',
                'date_initiated' => date('Y-m-d H:i:s')
            ]);

            return [
                'success' => true,
                'consultCallSid' => $consultCall->sid,
                'status' => 'consulting'
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }

    /**
     * Completes warm transfer - connects client with target user
     */
    public function completeTransferAction($callSid, $consultCallSid) {
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        try {
            // Get transfer state
            $transfer = $this->getTransferState($callSid);

            if (!$transfer || $transfer['consult_call_sid'] !== $consultCallSid) {
                throw new \Exception('Invalid transfer state');
            }

            // Connect original call to consultation call (conference)
            $conferenceName = 'transfer-' . $callSid . '-' . time();

            // Update original call to join conference
            $twilioClient->calls($callSid)->update([
                'twiml' => '<Response><Say>Connecting you now.</Say><Dial><Conference endConferenceOnExit="true">' . $conferenceName . '</Conference></Dial></Response>'
            ]);

            // Update consultation call to join conference
            $twilioClient->calls($consultCallSid)->update([
                'twiml' => '<Response><Dial><Conference>' . $conferenceName . '</Conference></Dial></Response>'
            ]);

            // Update transfer state
            $this->updateTransferState($callSid, ['status' => 'COMPLETED']);

            // Log transfer to timeline
            $this->logTransferToTimeline($callSid, $transfer['target_user_id']);

            return [
                'success' => true,
                'conference' => $conferenceName
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }

    private function saveTransferState($data) {
        global $DB;

        $DB->Query("
            INSERT INTO b_crm_call_transfers (
                ORIGINAL_CALL_SID,
                CONSULT_CALL_SID,
                TARGET_USER_ID,
                INITIATOR_USER_ID,
                STATUS,
                DATE_INITIATED
            ) VALUES (
                '{$DB->ForSql($data['original_call_sid'])}',
                '{$DB->ForSql($data['consult_call_sid'])}',
                {$data['target_user_id']},
                {$data['initiator_user_id']},
                '{$DB->ForSql($data['status'])}',
                '{$data['date_initiated']}'
            )
        ");
    }

    private function logTransferToTimeline($callSid, $targetUserId) {
        $targetUser = \CUser::GetByID($targetUserId)->Fetch();

        // Get deal from call
        $call = $this->getCallData($callSid);

        if ($call && $call['DEAL_ID']) {
            \Bitrix\Crm\Timeline\ActivityController::getInstance()->addNote([
                'OWNER_TYPE_ID' => \CCrmOwnerType::Deal,
                'OWNER_ID' => $call['DEAL_ID'],
                'AUTHOR_ID' => $GLOBALS['USER']->GetID(),
                'TEXT' => "Call transferred to {$targetUser['NAME']} {$targetUser['LAST_NAME']}",
                'SETTINGS' => [
                    'CALL_SID' => $callSid,
                    'TARGET_USER_ID' => $targetUserId
                ]
            ]);
        }
    }
}
```

```javascript
// React component for call transfer
class CallTransferDialog extends React.Component {
    constructor(props) {
        super(props);

        this.state = {
            targetUser: null,
            availableUsers: [],
            consulting: false,
            consultCallSid: null
        };
    }

    componentDidMount() {
        this.loadAvailableUsers();
    }

    loadAvailableUsers() {
        BX.ajax.runAction('crm:telephony.getAvailableUsers', {}).then((response) => {
            this.setState({ availableUsers: response.data.users });
        });
    }

    handleInitiateTransfer() {
        const { targetUser } = this.state;
        const { callSid } = this.props;

        // Warm transfer
        BX.ajax.runAction('crm:telephony.warmTransfer', {
            data: {
                callSid: callSid,
                targetUserId: targetUser.ID
            }
        }).then((response) => {
            if (response.data.success) {
                this.setState({
                    consulting: true,
                    consultCallSid: response.data.consultCallSid
                });
            }
        });
    }

    handleCompleteTransfer() {
        const { callSid } = this.props;
        const { consultCallSid } = this.state;

        BX.ajax.runAction('crm:telephony.completeTransfer', {
            data: {
                callSid: callSid,
                consultCallSid: consultCallSid
            }
        }).then((response) => {
            if (response.data.success) {
                BX.UI.Notification.Center.notify({
                    content: 'Transfer completed',
                    position: 'top-right'
                });
                this.props.onClose();
            }
        });
    }

    render() {
        const { targetUser, availableUsers, consulting } = this.state;

        if (consulting) {
            return (
                <div className="transfer-consulting">
                    <h3>Consulting with {targetUser.NAME}</h3>
                    <p>You are now speaking with the transfer target. The client is on hold.</p>
                    <div className="transfer-actions">
                        <button
                            className="btn btn-primary"
                            onClick={() => this.handleCompleteTransfer()}
                        >
                            Complete Transfer
                        </button>
                        <button
                            className="btn btn-secondary"
                            onClick={() => this.handleCancelTransfer()}
                        >
                            Cancel Transfer
                        </button>
                    </div>
                </div>
            );
        }

        return (
            <div className="transfer-dialog">
                <h3>Transfer Call</h3>

                <div className="user-selector">
                    <h4>Transfer to:</h4>
                    <select
                        value={targetUser?.ID || ''}
                        onChange={(e) => {
                            const user = availableUsers.find(u => u.ID == e.target.value);
                            this.setState({ targetUser: user });
                        }}
                    >
                        <option value="">Select user...</option>
                        {availableUsers.map(user => (
                            <option key={user.ID} value={user.ID}>
                                {user.NAME} {user.LAST_NAME} ({user.DEPARTMENT})
                            </option>
                        ))}
                    </select>
                </div>

                <div className="transfer-actions">
                    <button
                        className="btn btn-primary"
                        disabled={!targetUser}
                        onClick={() => this.handleInitiateTransfer()}
                    >
                        Start Consultation
                    </button>
                    <button
                        className="btn btn-secondary"
                        onClick={() => this.props.onClose()}
                    >
                        Cancel
                    </button>
                </div>
            </div>
        );
    }
}
```

### 3.2 Conference Calling

#### 3.2.1 Conference Flow

```
Dispatcher A talks with Client
    ↓
Dispatcher A clicks "Add to Conference"
    ↓
Client on hold
    ↓
Dispatcher A selects participants (Tech, Manager, other dispatchers)
    ↓
System calls all participants
    ↓
All join the conference
    ↓
Conference active - everyone hears each other
```

#### 3.2.2 Conference Implementation

```php
class ConferenceController {
    /**
     * Creates conference from current call
     */
    public function createConferenceAction($callSid, $participants) {
        global $USER;

        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        // Generate unique conference name
        $conferenceName = 'conf-' . $callSid . '-' . time();

        try {
            // Step 1: Add current call to conference
            $twilioClient->calls($callSid)->update([
                'twiml' => '<Response><Say>Creating conference. Please hold.</Say><Dial><Conference startConferenceOnEnter="true" endConferenceOnExit="true">' . $conferenceName . '</Conference></Dial></Response>'
            ]);

            // Step 2: Call all participants and add them to conference
            $participantCalls = [];

            foreach ($participants as $userId) {
                $user = \CUser::GetByID($userId)->Fetch();
                $userPhone = $user['PERSONAL_PHONE'];

                if (!$userPhone) {
                    continue;
                }

                $participantCall = $twilioClient->calls->create(
                    $userPhone,
                    COMPANY_PHONE,
                    [
                        'url' => SITE_URL . '/local/webhooks/twilio/conference_join.php?conferenceName=' . urlencode($conferenceName) . '&userName=' . urlencode($user['NAME']),
                        'statusCallback' => SITE_URL . '/local/webhooks/twilio/conference_status.php'
                    ]
                );

                $participantCalls[] = [
                    'user_id' => $userId,
                    'call_sid' => $participantCall->sid
                ];
            }

            // Save conference state
            $this->saveConference([
                'conference_name' => $conferenceName,
                'original_call_sid' => $callSid,
                'initiator_user_id' => $USER->GetID(),
                'participants' => json_encode($participantCalls),
                'date_created' => date('Y-m-d H:i:s')
            ]);

            return [
                'success' => true,
                'conferenceName' => $conferenceName,
                'participants' => $participantCalls
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }

    /**
     * Adds new participant to active conference
     */
    public function addParticipantAction($conferenceName, $userId) {
        $user = \CUser::GetByID($userId)->Fetch();
        $userPhone = $user['PERSONAL_PHONE'];

        if (!$userPhone) {
            return [
                'success' => false,
                'error' => 'User has no phone configured'
            ];
        }

        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        try {
            $call = $twilioClient->calls->create(
                $userPhone,
                COMPANY_PHONE,
                [
                    'url' => SITE_URL . '/local/webhooks/twilio/conference_join.php?conferenceName=' . urlencode($conferenceName) . '&userName=' . urlencode($user['NAME']),
                ]
            );

            return [
                'success' => true,
                'callSid' => $call->sid
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }

    /**
     * Removes participant from conference
     */
    public function removeParticipantAction($conferenceName, $participantCallSid) {
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        try {
            // Update participant's call to leave conference
            $twilioClient->calls($participantCallSid)->update([
                'status' => 'completed'
            ]);

            return [
                'success' => true
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }

    private function saveConference($data) {
        global $DB;

        $DB->Query("
            INSERT INTO b_crm_conferences (
                CONFERENCE_NAME,
                ORIGINAL_CALL_SID,
                INITIATOR_USER_ID,
                PARTICIPANTS,
                DATE_CREATED
            ) VALUES (
                '{$DB->ForSql($data['conference_name'])}',
                '{$DB->ForSql($data['original_call_sid'])}',
                {$data['initiator_user_id']},
                '{$DB->ForSql($data['participants'])}',
                '{$data['date_created']}'
            )
        ");
    }
}
```

```php
class CallHoldController {
    /**
     * Puts call on hold
     */
    public function holdCallAction($callSid) {
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        try {
            $call = $twilioClient->calls($callSid)->update([
                'twiml' => '<Response><Say>Please hold.</Say><Play loop="100">' . HOLD_MUSIC_URL . '</Play></Response>'
            ]);

            // Update call state
            global $DB;
            $DB->Query("
                UPDATE b_crm_call
                SET
                    UF_ON_HOLD = 'Y',
                    UF_HOLD_START = NOW()
                WHERE TWILIO_SID = '{$DB->ForSql($callSid)}'
            ");

            return [
                'success' => true,
                'status' => 'on_hold'
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }

    /**
     * Resumes call from hold
     */
    public function resumeCallAction($callSid) {
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        try {
            // Get original call TwiML or redirect to active conversation
            $call = $twilioClient->calls($callSid)->update([
                'twiml' => '<Response><Say>Thank you for holding.</Say><Dial></Dial></Response>'
            ]);

            // Update call state
            global $DB;
            $DB->Query("
                UPDATE b_crm_call
                SET
                    UF_ON_HOLD = 'N',
                    UF_HOLD_DURATION = TIMESTAMPDIFF(SECOND, UF_HOLD_START, NOW())
                WHERE TWILIO_SID = '{$DB->ForSql($callSid)}'
            ");

            return [
                'success' => true,
                'status' => 'active'
            ];

        } catch (\Exception $e) {
            return [
                'success' => false,
                'error' => $e->getMessage()
            ];
        }
    }
}
```

### 3.4 Mute/Unmute

#### 3.4.1 Mute Implementation

```javascript
// Client-side mute/unmute using Twilio Client SDK
class CallControlWidget extends React.Component {
    constructor(props) {
        super(props);

        this.state = {
            muted: false,
            onHold: false,
            device: null,
            activeConnection: null
        };
    }

    componentDidMount() {
        // Initialize Twilio Device
        this.initializeTwilioDevice();
    }

    async initializeTwilioDevice() {
        // Get capability token from backend
        const response = await BX.ajax.runAction('crm:telephony.getCapabilityToken', {});
        const token = response.data.token;

        const device = new Twilio.Device(token);

        device.on('ready', () => {
            console.log('Twilio Device ready');
        });

        device.on('connect', (connection) => {
            this.setState({ activeConnection: connection });
        });

        this.setState({ device });
    }

    handleMuteToggle() {
        const { activeConnection, muted } = this.state;

        if (!activeConnection) return;

        activeConnection.mute(!muted);
        this.setState({ muted: !muted });

        // Log mute action
        BX.ajax.runAction('crm:telephony.logMuteAction', {
            data: {
                callSid: activeConnection.parameters.CallSid,
                muted: !muted
            }
        });
    }

    handleHoldToggle() {
        const { callSid } = this.props;
        const { onHold } = this.state;

        if (onHold) {
            BX.ajax.runAction('crm:telephony.resumeCall', {
                data: { callSid }
            }).then(() => {
                this.setState({ onHold: false });
            });
        } else {
            BX.ajax.runAction('crm:telephony.holdCall', {
                data: { callSid }
            }).then(() => {
                this.setState({ onHold: true });
            });
        }
    }

    render() {
        const { muted, onHold } = this.state;

        return (
            <div className="call-controls">
                <button
                    className={`btn-control ${muted ? 'active' : ''}`}
                    onClick={() => this.handleMuteToggle()}
                    title="Mute/Unmute"
                >
                    {muted ? '🔇 Unmute' : '🔊 Mute'}
                </button>

                <button
                    className={`btn-control ${onHold ? 'active' : ''}`}
                    onClick={() => this.handleHoldToggle()}
                    title="Hold/Resume"
                >
                    {onHold ? '▶️ Resume' : '⏸️ Hold'}
                </button>

                <button
                    className="btn-control"
                    onClick={() => this.props.onTransfer()}
                    title="Transfer Call"
                >
                    ➡️ Transfer
                </button>

                <button
                    className="btn-control"
                    onClick={() => this.props.onConference()}
                    title="Add to Conference"
                >
                    👥 Conference
                </button>

                <button
                    className="btn-control btn-danger"
                    onClick={() => this.props.onHangup()}
                    title="Hang Up"
                >
                    📞 Hang Up
                </button>
            </div>
        );
    }
}
```

### 4.1 Database Schema

```sql
-- Enhanced call log table
CREATE TABLE b_crm_call_log (
    ID INT PRIMARY KEY AUTO_INCREMENT,

    -- Twilio data
    TWILIO_SID VARCHAR(50) UNIQUE NOT NULL,
    DIRECTION VARCHAR(10) NOT NULL,  -- 'INBOUND', 'OUTBOUND'

    -- Phone numbers
    FROM_NUMBER VARCHAR(20) NOT NULL,
    TO_NUMBER VARCHAR(20) NOT NULL,

    -- Status
    STATUS VARCHAR(20) NOT NULL,  -- 'COMPLETED', 'NO_ANSWER', 'BUSY', 'FAILED', 'CANCELED'

    -- Timing
    DATE_CREATE DATETIME NOT NULL,
    DATE_ANSWERED DATETIME,
    DATE_ENDED DATETIME,
    DURATION INT DEFAULT 0,  -- seconds

    -- CRM links
    CONTACT_ID INT,
    DEAL_ID INT,

    -- Call metadata
    UF_CALL_FLOW VARCHAR(50),  -- 'DIRECT', 'TRANSFERRED', 'CONFERENCE', 'VOICEMAIL'
    UF_AD_SOURCE VARCHAR(100),  -- From Job Source tracking
    UF_GCLID VARCHAR(255),

    -- Assignment
    ANSWERED_BY_USER_ID INT,

    -- Recording
    RECORDING_URL TEXT,
    RECORDING_DURATION INT,

    -- Tags
    TAGS TEXT,  -- JSON array: ["Spam", "Wrong Number", etc.]

    -- Notes
    NOTES TEXT,

    INDEX idx_date (DATE_CREATE),
    INDEX idx_status (STATUS),
    INDEX idx_direction (DIRECTION),
    INDEX idx_contact (CONTACT_ID),
    INDEX idx_deal (DEAL_ID),
    INDEX idx_user (ANSWERED_BY_USER_ID),
    INDEX idx_source (UF_AD_SOURCE),
    INDEX idx_twilio (TWILIO_SID),

    FOREIGN KEY (CONTACT_ID) REFERENCES b_crm_contact(ID) ON DELETE SET NULL,
    FOREIGN KEY (DEAL_ID) REFERENCES b_crm_deal(ID) ON DELETE SET NULL,
    FOREIGN KEY (ANSWERED_BY_USER_ID) REFERENCES b_user(ID) ON DELETE SET NULL
);

-- Full-text search index for notes
CREATE FULLTEXT INDEX idx_notes ON b_crm_call_log(NOTES);

-- Composite index for filtering
CREATE INDEX idx_filter ON b_crm_call_log (STATUS, DIRECTION, DATE_CREATE DESC);
```

```php
// Send real-time update when call is created/updated
class CallLogNotifier {
    /**
     * Notify all users about new call
     */
    public static function notifyNewCall($callData) {
        \Bitrix\Pull\Event::add(
            self::getAllDispatcherUserIds(),
            [
                'module_id' => 'crm',
                'command' => 'call_log_new',
                'params' => [
                    'call' => $callData
                ]
            ]
        );
    }

    /**
     * Notify all users about call update
     */
    public static function notifyCallUpdated($callData) {
        \Bitrix\Pull\Event::add(
            self::getAllDispatcherUserIds(),
            [
                'module_id' => 'crm',
                'command' => 'call_log_updated',
                'params' => [
                    'call' => $callData
                ]
            ]
        );
    }

    private static function getAllDispatcherUserIds() {
        global $DB;

        $userIds = [];
        $result = $DB->Query("
            SELECT ID
            FROM b_user
            WHERE UF_DEPARTMENT = 'DISPATCH'
              AND ACTIVE = 'Y'
        ");

        while ($row = $result->Fetch()) {
            $userIds[] = $row['ID'];
        }

        return $userIds;
    }
}
```

```javascript
class CallLogDashboard extends React.Component {
    constructor(props) {
        super(props);

        this.state = {
            calls: [],
            filters: {
                dateRange: 'last_7_days',
                status: 'all',
                direction: 'all',
                employee: 'all',
                source: 'all',
                search: ''
            },
            pagination: {
                page: 1,
                perPage: 50,
                total: 0
            },
            sorting: {
                field: 'DATE_CREATE',
                order: 'DESC'
            },
            selectedCalls: [],
            loading: false
        };
    }

    componentDidMount() {
        this.loadCalls();

        // Subscribe to real-time updates via Bitrix24 Pull & Push
        BX.PULL.subscribe({
            moduleId: 'crm',
            command: 'call_log_new',
            callback: (data) => {
                this.handleNewCall(data);
            }
        });

        BX.PULL.subscribe({
            moduleId: 'crm',
            command: 'call_log_updated',
            callback: (data) => {
                this.handleCallUpdated(data);
            }
        });
    }

    handleNewCall(data) {
        // Prepend new call to the list
        this.setState(prevState => ({
            calls: [data.call, ...prevState.calls],
            pagination: {
                ...prevState.pagination,
                total: prevState.pagination.total + 1
            }
        }));

        // Show notification
        BX.UI.Notification.Center.notify({
            content: `New ${data.call.DIRECTION} call: ${data.call.FROM_NUMBER}`,
            position: 'top-right'
        });
    }

    handleCallUpdated(data) {
        // Update specific call in the list
        this.setState(prevState => ({
            calls: prevState.calls.map(call =>
                call.ID === data.call.ID ? data.call : call
            )
        }));
    }

    loadCalls() {
        this.setState({ loading: true });

        BX.ajax.runAction('crm:calllog.list', {
            data: {
                filters: this.state.filters,
                pagination: this.state.pagination,
                sorting: this.state.sorting
            }
        }).then((response) => {
            this.setState({
                calls: response.data.items,
                pagination: {
                    ...this.state.pagination,
                    total: response.data.total
                },
                loading: false
            });
        });
    }

    handleFilterChange(filterName, value) {
        this.setState({
            filters: {
                ...this.state.filters,
                [filterName]: value
            },
            pagination: {
                ...this.state.pagination,
                page: 1  // Reset to first page
            }
        }, () => {
            this.loadCalls();
        });
    }

    handleCallBack(callId, phoneNumber) {
        BX.ajax.runAction('crm:telephony.initiateCall', {
            data: {
                toNumber: phoneNumber,
                callLogId: callId
            }
        }).then((response) => {
            if (response.data.success) {
                BX.UI.Notification.Center.notify({
                    content: 'Calling ' + phoneNumber + '...',
                    position: 'top-right'
                });
            }
        });
    }

    handleAddTag(callId) {
        BX.SidePanel.Instance.open('call-tag-dialog', {
            contentCallback: () => {
                return new Promise((resolve) => {
                    resolve('<div>Tag selection UI here</div>');
                });
            },
            events: {
                onClose: (event) => {
                    const selectedTags = event.slider.getData().get('tags');
                    if (selectedTags) {
                        this.saveTags(callId, selectedTags);
                    }
                }
            }
        });
    }

    saveTags(callId, tags) {
        BX.ajax.runAction('crm:calllog.saveTags', {
            data: {
                callId: callId,
                tags: tags
            }
        }).then(() => {
            this.loadCalls();  // Refresh list
        });
    }

    handleBulkTag() {
        const { selectedCalls } = this.state;

        if (selectedCalls.length === 0) {
            BX.UI.Notification.Center.notify({
                content: 'Please select calls to tag',
                position: 'top-right'
            });
            return;
        }

        // Show bulk tag dialog
        BX.SidePanel.Instance.open('bulk-tag-dialog', {
            width: 400,
            contentCallback: () => {
                return this.renderBulkTagDialog();
            }
        });
    }

    handleExportCSV() {
        window.location.href = '/local/api/calllog/export.php?' + new URLSearchParams({
            filters: JSON.stringify(this.state.filters)
        });
    }

    renderCallRow(call) {
        const statusIcon = this.getStatusIcon(call.STATUS);
        const duration = this.formatDuration(call.DURATION);

        return (
            <tr key={call.ID} className={`call-row status-${call.STATUS.toLowerCase()}`}>
                <td>
                    <input
                        type="checkbox"
                        checked={this.state.selectedCalls.includes(call.ID)}
                        onChange={(e) => this.toggleCallSelection(call.ID, e.target.checked)}
                    />
                </td>
                <td className="status">{statusIcon}</td>
                <td className="from">
                    <div className="phone">{call.FROM_NUMBER}</div>
                    <div className="contact-name">{call.CONTACT_NAME || 'Unknown'}</div>
                </td>
                <td className="to">
                    {call.ANSWERED_BY_NAME || '-'}
                </td>
                <td className="time">
                    <div className="duration">{duration}</div>
                    <div className="timestamp">{this.formatTime(call.DATE_CREATE)}</div>
                </td>
                <td className="flow">{call.UF_CALL_FLOW}</td>
                <td className="source">{call.UF_AD_SOURCE || '-'}</td>
                <td className="tags">
                    {this.renderTags(call.TAGS)}
                </td>
                <td className="actions">
                    {call.RECORDING_URL && (
                        <button
                            className="btn-icon"
                            onClick={() => this.playRecording(call.RECORDING_URL)}
                            title="Play Recording"
                        >
                            🎧
                        </button>
                    )}
                    <button
                        className="btn-icon"
                        onClick={() => this.handleCallBack(call.ID, call.FROM_NUMBER)}
                        title="Call Back"
                    >
                        📞
                    </button>
                    <button
                        className="btn-icon"
                        onClick={() => this.handleAddTag(call.ID)}
                        title="Add Tag"
                    >
                        🏷️
                    </button>
                    {call.DEAL_ID && (
                        <a
                            href={`/crm/deal/details/${call.DEAL_ID}/`}
                            className="deal-link"
                            title="View Deal"
                        >
                            💼 #{call.DEAL_ID}
                        </a>
                    )}
                </td>
            </tr>
        );
    }

    render() {
        const { calls, filters, pagination, loading } = this.state;

        return (
            <div className="call-log-dashboard">
                <div className="header">
                    <h2>Call Log</h2>
                </div>

                <div className="filters">
                    <select
                        value={filters.dateRange}
                        onChange={(e) => this.handleFilterChange('dateRange', e.target.value)}
                    >
                        <option value="today">Today</option>
                        <option value="yesterday">Yesterday</option>
                        <option value="last_7_days">Last 7 Days</option>
                        <option value="last_30_days">Last 30 Days</option>
                        <option value="custom">Custom Range</option>
                    </select>

                    <select
                        value={filters.status}
                        onChange={(e) => this.handleFilterChange('status', e.target.value)}
                    >
                        <option value="all">All Status</option>
                        <option value="COMPLETED">Completed</option>
                        <option value="NO_ANSWER">No Answer</option>
                        <option value="BUSY">Busy</option>
                        <option value="FAILED">Failed</option>
                    </select>

                    <select
                        value={filters.direction}
                        onChange={(e) => this.handleFilterChange('direction', e.target.value)}
                    >
                        <option value="all">All Directions</option>
                        <option value="INBOUND">Inbound</option>
                        <option value="OUTBOUND">Outbound</option>
                    </select>

                    <input
                        type="text"
                        placeholder="Search phone, name, notes..."
                        value={filters.search}
                        onChange={(e) => this.handleFilterChange('search', e.target.value)}
                    />
                </div>

                <div className="actions-bar">
                    <button onClick={() => this.handleExportCSV()}>Export CSV</button>
                    <button onClick={() => this.handleBulkTag()}>Bulk Tag</button>
                    <button onClick={() => this.clearFilters()}>Clear Filters</button>
                </div>

                <table className="call-log-table">
                    <thead>
                        <tr>
                            <th><input type="checkbox" onChange={(e) => this.toggleSelectAll(e.target.checked)} /></th>
                            <th>Status</th>
                            <th>From</th>
                            <th>To</th>
                            <th>Time</th>
                            <th>Flow</th>
                            <th>Source</th>
                            <th>Tags</th>
                            <th>Actions</th>
                        </tr>
                    </thead>
                    <tbody>
                        {loading ? (
                            <tr><td colSpan="9">Loading...</td></tr>
                        ) : calls.length === 0 ? (
                            <tr><td colSpan="9">No calls found</td></tr>
                        ) : (
                            calls.map(call => this.renderCallRow(call))
                        )}
                    </tbody>
                </table>

                <div className="pagination">
                    Showing {(pagination.page - 1) * pagination.perPage + 1}-
                    {Math.min(pagination.page * pagination.perPage, pagination.total)} of {pagination.total}
                    <div className="pagination-controls">
                        {this.renderPaginationButtons()}
                    </div>
                </div>
            </div>
        );
    }

    getStatusIcon(status) {
        const icons = {
            'COMPLETED': '✓',
            'NO_ANSWER': '✗',
            'BUSY': '⊘',
            'FAILED': '⚠',
            'CANCELED': '⊗'
        };
        return icons[status] || '?';
    }

    formatDuration(seconds) {
        if (!seconds) return '0s';
        const minutes = Math.floor(seconds / 60);
        const secs = seconds % 60;
        return minutes > 0 ? `${minutes}m ${secs}s` : `${secs}s`;
    }

    formatTime(dateStr) {
        const date = new Date(dateStr);
        return date.toLocaleString('en-US', {
            hour: 'numeric',
            minute: '2-digit',
            hour12: true
        });
    }

    renderTags(tagsJson) {
        if (!tagsJson) return null;

        const tags = JSON.parse(tagsJson);
        return (
            <div className="tags-list">
                {tags.map((tag, idx) => (
                    <span key={idx} className="tag">{tag}</span>
                ))}
            </div>
        );
    }
}
```

### 4.4 Call Log Backend API

```php
class CallLogController {
    /**
     * Get call log list with filtering
     */
    public function listAction($filters, $pagination, $sorting) {
        global $DB;

        // Build WHERE clause
        $where = ['1=1'];

        if ($filters['dateRange'] !== 'all') {
            $dateCondition = $this->buildDateCondition($filters['dateRange']);
            $where[] = $dateCondition;
        }

        if ($filters['status'] !== 'all') {
            $where[] = "STATUS = '{$DB->ForSql($filters['status'])}'";
        }

        if ($filters['direction'] !== 'all') {
            $where[] = "DIRECTION = '{$DB->ForSql($filters['direction'])}'";
        }

        if ($filters['employee'] !== 'all') {
            $where[] = "ANSWERED_BY_USER_ID = " . (int)$filters['employee'];
        }

        if ($filters['source'] !== 'all') {
            $where[] = "UF_AD_SOURCE = '{$DB->ForSql($filters['source'])}'";
        }

        if (!empty($filters['search'])) {
            $search = $DB->ForSql($filters['search']);
            $where[] = "(
                FROM_NUMBER LIKE '%{$search}%'
                OR TO_NUMBER LIKE '%{$search}%'
                OR NOTES LIKE '%{$search}%'
                OR CONTACT_NAME LIKE '%{$search}%'
            )";
        }

        $whereClause = implode(' AND ', $where);

        // Calculate pagination
        $offset = ($pagination['page'] - 1) * $pagination['perPage'];
        $limit = (int)$pagination['perPage'];

        // Get total count
        $countResult = $DB->Query("
            SELECT COUNT(*) as total
            FROM b_crm_call_log cl
            LEFT JOIN b_crm_contact c ON cl.CONTACT_ID = c.ID
            WHERE {$whereClause}
        ")->Fetch();

        $total = $countResult['total'];

        // Get calls
        $calls = [];
        $result = $DB->Query("
            SELECT
                cl.*,
                CONCAT(c.NAME, ' ', c.LAST_NAME) as CONTACT_NAME,
                CONCAT(u.NAME, ' ', u.LAST_NAME) as ANSWERED_BY_NAME
            FROM b_crm_call_log cl
            LEFT JOIN b_crm_contact c ON cl.CONTACT_ID = c.ID
            LEFT JOIN b_user u ON cl.ANSWERED_BY_USER_ID = u.ID
            WHERE {$whereClause}
            ORDER BY {$sorting['field']} {$sorting['order']}
            LIMIT {$limit} OFFSET {$offset}
        ");

        while ($row = $result->Fetch()) {
            $calls[] = $row;
        }

        return [
            'items' => $calls,
            'total' => $total
        ];
    }

    /**
     * Save tags to call
     */
    public function saveTagsAction($callId, $tags) {
        global $DB;

        $tagsJson = json_encode($tags);

        $DB->Query("
            UPDATE b_crm_call_log
            SET TAGS = '{$DB->ForSql($tagsJson)}'
            WHERE ID = {$callId}
        ");

        return ['success' => true];
    }

    /**
     * Export call log to CSV
     */
    public function exportAction($filters) {
        // Get all calls matching filters (no pagination)
        $calls = $this->getCallsForExport($filters);

        // Generate CSV
        header('Content-Type: text/csv');
        header('Content-Disposition: attachment; filename="call_log_' . date('Y-m-d') . '.csv"');

        $output = fopen('php://output', 'w');

        // Headers
        fputcsv($output, [
            'Date',
            'Time',
            'Direction',
            'From',
            'To',
            'Duration',
            'Status',
            'Call Flow',
            'Ad Source',
            'Answered By',
            'Deal ID',
            'Tags',
            'Recording URL'
        ]);

        // Data
        foreach ($calls as $call) {
            fputcsv($output, [
                date('Y-m-d', strtotime($call['DATE_CREATE'])),
                date('H:i:s', strtotime($call['DATE_CREATE'])),
                $call['DIRECTION'],
                $call['FROM_NUMBER'],
                $call['TO_NUMBER'],
                $call['DURATION'],
                $call['STATUS'],
                $call['UF_CALL_FLOW'],
                $call['UF_AD_SOURCE'],
                $call['ANSWERED_BY_NAME'],
                $call['DEAL_ID'],
                $call['TAGS'],
                $call['RECORDING_URL']
            ]);
        }

        fclose($output);
        exit;
    }

    private function buildDateCondition($dateRange) {
        switch ($dateRange) {
            case 'today':
                return "DATE(DATE_CREATE) = CURDATE()";
            case 'yesterday':
                return "DATE(DATE_CREATE) = DATE_SUB(CURDATE(), INTERVAL 1 DAY)";
            case 'last_7_days':
                return "DATE_CREATE >= DATE_SUB(NOW(), INTERVAL 7 DAY)";
            case 'last_30_days':
                return "DATE_CREATE >= DATE_SUB(NOW(), INTERVAL 30 DAY)";
            default:
                return "1=1";
        }
    }
}
```

## 5. Cost Analysis

### 5.1 Call Control Costs

```
Twilio Conference:
- Conference participant: same as regular call ($0.013/min)
- No additional conference fees

Call Transfer:
- Warm transfer consultation: 2 simultaneous calls during consultation
- Average consultation: 30 seconds
- Additional cost: ~$0.0065 per transfer

Call Hold:
- No additional cost (music playback included)

Estimated monthly costs:
- Transfers: ~100/month × $0.0065 = $0.65/month
- Conferences: ~50/month × 5 min × 3 participants × $0.013 = $9.75/month

Total: ~$10.40/month (~$125/year)
```

### 5.2 Call Log Storage

```
Database storage:
- ~500 bytes per call record
- 90,000 calls/month × 500 bytes = 45 MB/month
- 12 months: 540 MB

Negligible cost - included in Bitrix24 database
```
