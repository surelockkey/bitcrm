**Epic:** Payment Processing System
**Component:** Service Fee Calculation & Application
**Version:** 1.0
**Last Updated:** January 16, 2025

## 1. System Overview

### 1.1. Purpose

Implement automatic service fee calculation and application for credit card payments in Bitrix24. When a client pays by credit card, the system automatically adds a configurable service fee to the invoice total to cover payment processing costs.

**Key Features:**
- Automatic service fee calculation based on payment method
- Fixed 3% fee for all credit card payments (all states)
- Transparent display to client before payment
- Automatic addition to invoice as separate line item
- Internal technician commission calculation adjustment

### 1.2. Business Requirements

**Why Service Fees?**

Credit card processing incurs costs to the business:
- Stripe fee: 2.9% + $0.30 per transaction
- Additional costs for international cards
- Chargeback fees: $15 per dispute

**Service Fee Policy:**
- **Applied to:** Credit Card payments only
- **Rate:** Fixed 3% for all states
- **NOT applied to:** Cash, Check, ACH, Cash by Referral
- **Visibility:** Shown to client BEFORE payment confirmation

**Example Calculation:**

```
Invoice Subtotal:   $500.00
Tax (7%):           $35.00
──────────────────────────
Subtotal with Tax:  $535.00

[Client selects: Credit Card]
→ System automatically adds:
Service Fee (3%):   $16.05
──────────────────────────
Final Total:        $551.05
```

```
│                   SERVICE FEE FLOW                   │
└──────────────────────────────────────────────────────┘

  │

  ↓
```


### 2.2. Automatic Fee Calculation

**Trigger Points:**

1. **Client Portal - Payment Method Selection**
   - Client opens invoice link
   - Client selects "Credit Card" as payment method
   - JavaScript calculates fee in real-time
   - Updated total displayed before Stripe redirect

2. **Dispatcher Sending Invoice**
   - Dispatcher selects "Credit Card" for payment
   - System pre-calculates fee
   - Stripe Payment Link created with fee included

3. **Technician Tap to Pay**
   - Technician initiates Tap to Pay
   - System automatically adds service fee
   - Total displayed on iPhone before payment

**Calculation Logic:**

```php
// File: /local/php_interface/include/service_fee_calculator.php

namespace Local\Payment;

class ServiceFeeCalculator {

    /**
     * Calculate service fee for a Deal
     *
     * @param int $dealId Deal ID
     * @param string $paymentMethod Payment method (e.g., 'credit_card')
     * @return array ['fee' => float, 'rate' => float]
     */
    public static function calculateFee($dealId, $paymentMethod) {
        // Check if service fee applies to this payment method
        if (!self::isServiceFeeApplicable($paymentMethod)) {
            return ['fee' => 0, 'rate' => 0];
        }

        // Get invoice subtotal (with tax)
        $subtotal = self::getInvoiceSubtotal($dealId);

        // Get service fee rate (fixed 3% for all states)
        $rate = \COption::GetOptionFloat('service_fee', 'rate', 3.0);

        // Calculate fee
        $fee = $subtotal * ($rate / 100);

        // Round to 2 decimal places
        $fee = round($fee, 2);

        return [
            'fee' => $fee,
            'rate' => $rate
        ];
    }

    /**
     * Check if service fee applies to payment method
     */
    private static function isServiceFeeApplicable($paymentMethod) {
        $config = \COption::GetOptionString('service_fee', 'enabled_methods', 'credit_card,debit_card');
        $enabledMethods = explode(',', $config);

        return in_array($paymentMethod, $enabledMethods);
    }

    /**
     * Get invoice subtotal (including tax, excluding existing service fee)
     */
    private static function getInvoiceSubtotal($dealId) {
        global $DB;

        // Get all line items except service fee
        $result = $DB->Query("
            SELECT SUM(PRICE * QUANTITY) as subtotal
            FROM b_sale_basket
            WHERE ORDER_ID = (
                SELECT ID FROM b_sale_order WHERE ACCOUNT_NUMBER = '{$dealId}'
            )
            AND NAME NOT LIKE '%Service Fee%'
        ")->Fetch();

        return floatval($result['subtotal'] ?? 0);
    }

    /**
     * Apply service fee to invoice
     * Adds service fee as a line item to the order basket
     */
    public static function applyServiceFee($dealId, $paymentMethod) {
        // Calculate fee
        $feeData = self::calculateFee($dealId, $paymentMethod);

        if ($feeData['fee'] <= 0) {
            return; // No fee to apply
        }

        // Get order
        $order = \Bitrix\Sale\Order::loadByAccountNumber($dealId);
        if (!$order) {
            throw new \Exception('Order not found');
        }

        // Check if service fee already exists
        $basket = $order->getBasket();
        foreach ($basket as $item) {
            if (strpos($item->getField('NAME'), 'Service Fee') !== false) {
                // Service fee already added, update amount
                $item->setField('PRICE', $feeData['fee']);
                $item->save();
                $order->save();
                return;
            }
        }

        // Add service fee as new line item
        $item = $basket->createItem('catalog', 0);
        $item->setFields([
            'NAME' => sprintf('Service Fee (%s%%)', $feeData['rate']),
            'PRICE' => $feeData['fee'],
            'QUANTITY' => 1,
            'CUSTOM_PRICE' => 'Y',
            'CURRENCY' => 'USD',
            'LID' => SITE_ID,
            'PRODUCT_PROVIDER_CLASS' => '\Bitrix\Sale\BasketItem'
        ]);

        $order->save();

        // Update Deal custom field
        \CCrmDeal::Update($dealId, [
            'UF_CRM_SERVICE_FEE' => $feeData['fee']
        ]);

        // Add timeline entry
        \Bitrix\Crm\Timeline\CommentController::getInstance()->onCreate([
            'ENTITY_TYPE_ID' => \CCrmOwnerType::Deal,
            'ENTITY_ID' => $dealId,
            'COMMENT' => sprintf(
                "💳 Service fee added: $%s (%s%% of $%s)",
                number_format($feeData['fee'], 2),
                $feeData['rate'],
                number_format(self::getInvoiceSubtotal($dealId), 2)
            )
        ]);
    }

    /**
     * Remove service fee from invoice
     * Used when payment method changes from CC to ACH/Cash
     */
    public static function removeServiceFee($dealId) {
        $order = \Bitrix\Sale\Order::loadByAccountNumber($dealId);
        if (!$order) {
            return;
        }

        $basket = $order->getBasket();
        foreach ($basket as $item) {
            if (strpos($item->getField('NAME'), 'Service Fee') !== false) {
                $item->delete();
                break;
            }
        }

        $order->save();

        // Clear Deal field
        \CCrmDeal::Update($dealId, [
            'UF_CRM_SERVICE_FEE' => 0
        ]);
    }
}
```

```javascript
            if (data.success) {
                showServiceFee(data.fee, data.rate);
                updateTotal(invoiceSubtotal + data.fee);
            }
        })
        .catch(error => {
            console.error('Error calculating service fee:', error);
        });
    }

        const serviceFeeRow = document.getElementById('service-fee-row');

        // Show explanation
        document.getElementById('service-fee-explanation').style.display = 'block';

        document.getElementById('service-fee-row').style.display = 'none';
        document.getElementById('service-fee-explanation').style.display = 'none';
    }

    function updateTotal(newTotal) {
        document.getElementById('invoice-total').textContent = '$' + newTotal.toFixed(2);
        document.getElementById('payment-amount').value = newTotal.toFixed(2);
    }
```

### 2.4. Stripe Payment Intent Integration

**Create Payment Intent with Service Fee:**

```php
// File: /local/php_interface/include/stripe_payment_intent.php

namespace Local\Payment;

use Local\Payment\ServiceFeeCalculator;

class StripePaymentIntent {

    /**
     * Create Stripe Payment Intent with service fee
     */
    public static function createPaymentIntent($dealId, $paymentMethod) {
        // Get invoice total
        $invoiceTotal = self::getInvoiceTotal($dealId);

        // Apply service fee if credit card
        if ($paymentMethod === 'card') {
            ServiceFeeCalculator::applyServiceFee($dealId, $paymentMethod);

            // Recalculate total with service fee
            $invoiceTotal = self::getInvoiceTotal($dealId);
        }

        // Create Stripe Payment Intent
        \Stripe\Stripe::setApiKey(\COption::GetOptionString('stripe', 'secret_key'));

        $paymentIntent = \Stripe\PaymentIntent::create([
            'amount' => intval($invoiceTotal * 100), // Convert to cents
            'currency' => 'usd',
            'payment_method_types' => [$paymentMethod],
            'metadata' => [
                'bitrix_deal_id' => $dealId,
                'service_area' => self::getStateCode($dealId)
            ],
            'description' => "Invoice for Deal #$dealId"
        ]);

        // Save Payment Intent ID to Deal
        \CCrmDeal::Update($dealId, [
            'UF_CRM_STRIPE_PI_ID' => $paymentIntent->id
        ]);

        return $paymentIntent;
    }

    private static function getInvoiceTotal($dealId) {
        $order = \Bitrix\Sale\Order::loadByAccountNumber($dealId);
        if (!$order) {
            throw new \Exception('Order not found');
        }

        return $order->getPrice();
    }

    private static function getStateCode($dealId) {
        $deal = \CCrmDeal::GetByID($dealId);
        return $deal['UF_CRM_SERVICE_AREA'] ?? 'TX';
    }
}
```

## 3. Technician Commission Calculation

### 3.1. Internal CC Fee Field

**Purpose:**

The `Technician CC Fee (%)` field is an **internal-only** field used in commission calculations. It's NOT displayed to clients.

**Usage in Commission Formula:**

```
Tech Profit = (Total - Tax - Parts - Company Parts - (Credit × Credit Card Fee %)) × Tech Share %

Where:
  Total = Invoice total (including service fee charged to client)
  Credit = Amount paid by credit card
  Credit Card Fee % = Internal technician CC fee percentage
  Tech Share % = Technician commission percentage (e.g., 50%)
```

**Example:**

```
Invoice:
  Services:        $500.00
  Tax:             $35.00
  Service Fee:     $16.05 (charged to client)
  ─────────────────────
  Total Paid:      $551.05

Commission Calculation:
  Total:           $551.05
  Tax:             -$35.00
  Parts:           -$0.00
  CC Fee (3%):     -$16.05  ← Deducted from tech commission
  ─────────────────────
  Taxable:         $500.00
  Tech Share (50%): $250.00
```

**Configuration:**

```php
// Field: UF_CRM_TECH_CC_FEE
// Default: 3.0%
// Editable by: Admin only
// Visible in: Reports only (not client-facing)

// Update Deal with technician CC fee
\CCrmDeal::Update($dealId, [
    'UF_CRM_TECH_CC_FEE' => 3.0
]);
```

```php
// File: /local/php_interface/reports/commission_report.php

function calculateTechnicianCommission($dealId) {
    $deal = \CCrmDeal::GetByID($dealId);

    // Get invoice data
    $total = floatval($deal['OPPORTUNITY']);
    $tax = floatval($deal['UF_CRM_TAX_AMOUNT']);
    $parts = floatval($deal['UF_CRM_PARTS_COST']);
    $companyParts = floatval($deal['UF_CRM_COMPANY_PARTS']);

    // Get payment method
    $paymentMethod = $deal['UF_CRM_PAYMENT_METHOD'];

    // Calculate CC fee deduction if paid by credit card
    $ccFeeDeduction = 0;
    if ($paymentMethod === 'Credit Card') {
        $creditAmount = $total; // Assuming full amount paid by card
        $ccFeePercent = floatval($deal['UF_CRM_TECH_CC_FEE'] ?? 3.0);
        $ccFeeDeduction = $creditAmount * ($ccFeePercent / 100);
    }

    // Calculate taxable amount
    $taxableAmount = $total - $tax - $parts - $companyParts - $ccFeeDeduction;

    // Get tech share percentage
    $techSharePercent = floatval($deal['UF_CRM_TECH_SHARE'] ?? 50.0);

    // Calculate commission
    $commission = $taxableAmount * ($techSharePercent / 100);

    return [
        'total' => $total,
        'tax' => $tax,
        'parts' => $parts,
        'company_parts' => $companyParts,
        'cc_fee_deduction' => $ccFeeDeduction,
        'taxable_amount' => $taxableAmount,
        'tech_share_percent' => $techSharePercent,
        'commission' => $commission
    ];
}
```

## 4. Event Handlers

### 4.1. Payment Method Change Handler

**Trigger:** When payment method changes on Deal

```php
// File: /local/php_interface/init.php

use Bitrix\Main\EventManager;
use Local\Payment\ServiceFeeCalculator;

$eventManager = EventManager::getInstance();

// Listen for Deal field updates
$eventManager->addEventHandler(
    'crm',
    'OnAfterCrmDealUpdate',
    'handlePaymentMethodChange'
);

function handlePaymentMethodChange($event) {
    $fields = $event->getParameter('FIELDS');
    $dealId = $fields['ID'];

    // Check if payment method changed
    if (isset($fields['UF_CRM_PAYMENT_METHOD'])) {
        $paymentMethod = $fields['UF_CRM_PAYMENT_METHOD'];

        // Apply or remove service fee based on payment method
        if ($paymentMethod === 'Credit Card') {
            ServiceFeeCalculator::applyServiceFee($dealId, 'credit_card');
        } else {
            ServiceFeeCalculator::removeServiceFee($dealId);
        }
    }
}
```

### 4.2. Invoice Generation Hook

**Trigger:** When invoice is generated

```php
// File: /local/php_interface/include/invoice_hooks.php

use Bitrix\Main\EventManager;
use Local\Payment\ServiceFeeCalculator;

EventManager::getInstance()->addEventHandler(
    'sale',
    'OnSaleOrderSaved',
    'handleInvoiceGeneration'
);

function handleInvoiceGeneration($event) {
    $order = $event->getParameter('ENTITY');
    $dealId = $order->getField('ACCOUNT_NUMBER');

    // Get payment method from Deal
    $deal = \CCrmDeal::GetByID($dealId);
    $paymentMethod = $deal['UF_CRM_PAYMENT_METHOD'] ?? 'Credit Card';

    // If credit card and no service fee yet, apply it
    if ($paymentMethod === 'Credit Card') {
        $basket = $order->getBasket();
        $hasServiceFee = false;

        foreach ($basket as $item) {
            if (strpos($item->getField('NAME'), 'Service Fee') !== false) {
                $hasServiceFee = true;
                break;
            }
        }

        if (!$hasServiceFee) {
            ServiceFeeCalculator::applyServiceFee($dealId, 'credit_card');
        }
    }
}
```
