# TR-04: Call Recording & Monitoring

**Epic:** Telephony Integration
**Document Version:** 1.0
**Date:** 2025-10-10
**Status:** Draft

## 1. Overview

### 1.1 Purpose
Ensure automatic recording of all calls (inbound, outbound, masked) with playback capability in Bitrix24 CRM, real-time supervisor monitoring, and recording storage for quality assurance and legal compliance.

### 1.2 Scope
- Automatic recording of all calls through Twilio
- Storage of recordings in S3-compatible cloud storage
- Playback of recordings in Bitrix24 timeline
- Supervisor live monitoring (listen, whisper, barge)
- Call quality metrics (MOS score, latency, jitter)
- Retention policy (2 years storage)
- PCI compliance (pause recording for payment info)

## 2. Architecture

### 2.1 Recording Flow
```
┌──────────────────────────────────────────────────────────────┐
│                      Call Recording Flow                      │
└──────────────────────────────────────────────────────────────┘

1. INBOUND CALL:
   Twilio → record-from-answer → Bitrix24 (dispatcher answers)
                    ↓
            Recording starts immediately
                    ↓
            Call ends → Twilio saves to S3
                    ↓
            Webhook: recordingStatusCallback
                    ↓
            Bitrix24 saves recording URL to Deal

2. OUTBOUND CALL (Dispatcher):
   Bitrix24 → Twilio (record=true) → Client
                    ↓
            Recording starts when client answers
                    ↓
            Same flow as inbound

3. MASKED CALL (Tech ↔ Client):
   Tech dials extension → Twilio Function
                    ↓
            Twilio connects with record=true
                    ↓
            Recording saved to Deal
```

### 2.2 Storage Architecture (Hybrid Approach)

**Strategy:** Use Twilio Storage for the first 30 days (free + fast CDN access), then automatically migrate to S3-compatible storage for long-term storage.

```
┌──────────────────────────────────────────────────────────────────┐
│               Hybrid Recording Storage Strategy                  │
└──────────────────────────────────────────────────────────────────┘

📅 Day 0-30: TWILIO STORAGE (Hot Storage)
┌─────────────────┐
│  Twilio CDN     │  ← Recording created here
│  (Hot Storage)  │     ✅ FREE first 30 days
│                 │     ✅ Fast access (CDN globally distributed)
└─────────────────┘     ✅ High availability
       │
       │  📦 Day 30: Auto-migration (cron job)
       ▼
┌─────────────────┐
│ S3-Compatible   │  ← Migration after 30 days
│ (Cold Storage)  │     ✅ Backblaze B2: $0.006/GB/month
│ Backblaze B2    │     ✅ Wasabi: $0.0059/GB/month
└─────────────────┘     ✅ Long-term storage (up to 2 years)
       │
       │  🔗 Bitrix24 dynamically updates URL
       ▼
┌─────────────────┐
│   Bitrix24      │
│ RECORDING_URL   │  → Day 0-30:  https://api.twilio.com/RE...
│   (Dynamic)     │  → Day 31+:   https://s3.backblaze.com/...
└─────────────────┘
```

**Advantages of Hybrid Approach:**
1. ✅ **Fast access to recent recordings** (0-30 days via Twilio CDN)
2. ✅ **FREE first 30 days** (most popular period for listening)
3. ✅ **Cheap long-term storage** ($0.006/GB vs $0.0025/min)
4. ✅ **Automatic migration** without manual intervention
5. ✅ **Seamless playback** - user doesn't notice the difference

**Why exactly 30 days?**
- Statistics show: **80-90% of recordings** are listened to within the first 30 days
- After 30 days Twilio starts charging $0.0025/min for storage
- Migration on day 30 maximizes savings

**Storage Options for Cold Storage:**
- **Backblaze B2**: $0.006/GB/month ✅✅ (recommended)
- **Wasabi**: $0.0059/GB/month + no egress fees ✅✅
- **AWS S3**: $0.023/GB/month (more expensive)
- **Cloudflare R2**: $0.015/GB/month + zero egress fees

### 3.1 Automatic Call Recording

#### 3.1.1 Inbound Calls Recording
```php
// TR-01 modification: add recording to inbound call handler
function handleInboundCall($twilioRequest) {
    $twiml = new VoiceResponse();

    // Enqueue to dispatchers with recording enabled
    $dial = $twiml->dial([
        'action' => 'https://your-bitrix.com/rest/twilio/call-status',
        'record' => 'record-from-answer',  // Start recording when answered
        'recordingStatusCallback' => 'https://your-bitrix.com/rest/twilio/recording-complete',
        'recordingStatusCallbackMethod' => 'POST',
        'recordingStatusCallbackEvent' => ['completed', 'absent'],
        'timeout' => 120
    ]);

    $dial->queue('DispatcherQueue', [
        'url' => 'https://your-bitrix.com/rest/twilio/queue-wait-music'
    ]);

    return $twiml;
}
```

**Twilio Parameters:**
- `record-from-answer`: starts recording when dispatcher answers (doesn't record ring tone)
- `record-from-ringing`: starts recording from first ring (records hold music)
- `do-not-record`: disable recording (not used, we record everything)

### 3.2 Recording Storage & Webhook Handler

#### 3.2.1 Twilio Recording Webhook
```php
// /rest/twilio/recording-complete
class TwilioRecordingWebhook {
    public function recordingCompleteAction(Request $request) {
        // Validate Twilio signature
        $validator = new RequestValidator(TWILIO_AUTH_TOKEN);
        if (!$validator->validate(
            $request->headers->get('X-Twilio-Signature'),
            $request->getUri(),
            $request->request->all()
        )) {
            return new Response('Forbidden', 403);
        }

        $data = [
            'RecordingSid' => $request->get('RecordingSid'),
            'RecordingUrl' => $request->get('RecordingUrl'),
            'RecordingDuration' => $request->get('RecordingDuration'),  // seconds
            'CallSid' => $request->get('CallSid'),
            'RecordingStatus' => $request->get('RecordingStatus'),  // completed, absent
            'RecordingChannels' => $request->get('RecordingChannels'),  // 1=mono, 2=stereo
        ];

        if ($data['RecordingStatus'] === 'absent') {
            \Bitrix\Main\Diag\Logger::log(
                "Recording absent for CallSid: {$data['CallSid']}",
                'RECORDING_ERROR'
            );
            return new Response('OK', 200);
        }

        // Find Call in Bitrix24
        $call = $this->findCallByTwilioSid($data['CallSid']);

        if (!$call) {
            \Bitrix\Main\Diag\Logger::log(
                "Call not found for CallSid: {$data['CallSid']}",
                'RECORDING_ERROR'
            );
            return new Response('OK', 200);
        }

        // Download recording from Twilio and upload to S3
        $s3Url = $this->downloadAndUploadToS3($data);

        // Save to database
        $this->saveRecording([
            'CALL_ID' => $call['ID'],
            'ENTITY_TYPE' => $call['ENTITY_TYPE'],
            'ENTITY_ID' => $call['ENTITY_ID'],
            'TWILIO_SID' => $data['RecordingSid'],
            'RECORDING_URL' => $s3Url,
            'DURATION' => $data['RecordingDuration'],
            'CHANNELS' => $data['RecordingChannels'],
            'FILE_SIZE' => $this->getFileSize($s3Url),
            'DATE_CREATE' => new DateTime()
        ]);

        // Add to CRM timeline
        $this->addRecordingToTimeline($call, $s3Url, $data['RecordingDuration']);

        return new Response('OK', 200);
    }

    private function downloadAndUploadToS3($recordingData) {
        // Download from Twilio (authenticated)
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);
        $recordingUrl = $recordingData['RecordingUrl'] . '.mp3';

        $recordingContent = file_get_contents($recordingUrl, false, stream_context_create([
            'http' => [
                'header' => "Authorization: Basic " . base64_encode(TWILIO_SID . ':' . TWILIO_TOKEN)
            ]
        ]));

        // Upload to AWS S3
        $s3Client = new Aws\S3\S3Client([
            'version' => 'latest',
            'region' => AWS_REGION,
            'credentials' => [
                'key' => AWS_ACCESS_KEY,
                'secret' => AWS_SECRET_KEY
            ]
        ]);

        $fileName = 'recordings/' . date('Y/m/d') . '/' . $recordingData['RecordingSid'] . '.mp3';

        $s3Client->putObject([
            'Bucket' => AWS_S3_BUCKET,
            'Key' => $fileName,
            'Body' => $recordingContent,
            'ContentType' => 'audio/mpeg',
            'ServerSideEncryption' => 'AES256',  // Encrypt at rest
            'Metadata' => [
                'CallSid' => $recordingData['CallSid'],
                'Duration' => $recordingData['RecordingDuration']
            ]
        ]);

        // Generate presigned URL (valid for 7 days)
        $s3Url = $s3Client->getObjectUrl(AWS_S3_BUCKET, $fileName);

        return $s3Url;
    }

    private function saveRecording($data) {
        global $DB;

        $DB->Query("
            INSERT INTO b_crm_call_recording (
                CALL_ID, ENTITY_TYPE, ENTITY_ID,
                TWILIO_SID, RECORDING_URL, DURATION,
                CHANNELS, FILE_SIZE, DATE_CREATE
            ) VALUES (
                {$data['CALL_ID']},
                '{$data['ENTITY_TYPE']}',
                {$data['ENTITY_ID']},
                '{$data['TWILIO_SID']}',
                '{$DB->ForSql($data['RECORDING_URL'])}',
                {$data['DURATION']},
                {$data['CHANNELS']},
                {$data['FILE_SIZE']},
                NOW()
            )
        ");
    }

    private function addRecordingToTimeline($call, $recordingUrl, $duration) {
        \Bitrix\Crm\Timeline\ActivityController::getInstance()->onCreate(
            $call['ENTITY_ID'],
            [
                'TYPE_ID' => \CCrmActivityType::Call,
                'SUBJECT' => 'Call Recording (' . gmdate('i:s', $duration) . ')',
                'DESCRIPTION' => 'Recording available',
                'COMMUNICATIONS' => [[
                    'ENTITY_TYPE_ID' => \CCrmOwnerType::Contact,
                    'VALUE' => $call['PHONE'],
                    'ENTITY_ID' => $call['CONTACT_ID']
                ]],
                'SETTINGS' => [
                    'RECORDING_URL' => $recordingUrl,
                    'DURATION' => $duration
                ]
            ]
        );
    }
}
```

### 3.3 30-Day Migration (Twilio → S3)

#### 3.3.1 Cron Job: Daily Migration Check
```php
// Runs daily at 2 AM: check recordings older than 30 days
class RecordingMigrationCron {
    public function execute() {
        $cutoffDate = date('Y-m-d', strtotime('-30 days'));

        global $DB;

        // Get recordings that are exactly 30 days old and still on Twilio
        $recordings = $DB->Query("
            SELECT ID, TWILIO_SID, RECORDING_URL, ENTITY_TYPE, ENTITY_ID, DURATION
            FROM b_crm_call_recording
            WHERE DATE_CREATE < '{$cutoffDate}'
              AND RECORDING_URL LIKE '%twilio.com%'
              AND MIGRATED_TO_S3 = 'N'
            LIMIT 100
        ");

        $migratedCount = 0;

        while ($recording = $recordings->Fetch()) {
            try {
                // Download from Twilio
                $recordingData = $this->downloadFromTwilio($recording['TWILIO_SID']);

                // Upload to S3
                $s3Url = $this->uploadToS3($recordingData, $recording);

                // Update database with new S3 URL
                $DB->Query("
                    UPDATE b_crm_call_recording
                    SET
                        RECORDING_URL = '{$DB->ForSql($s3Url)}',
                        MIGRATED_TO_S3 = 'Y',
                        DATE_MIGRATED = NOW()
                    WHERE ID = {$recording['ID']}
                ");

                // Delete from Twilio to avoid storage charges
                $this->deleteFromTwilio($recording['TWILIO_SID']);

                $migratedCount++;

                \Bitrix\Main\Diag\Logger::log(
                    "Recording {$recording['ID']} migrated: Twilio → S3",
                    'RECORDING_MIGRATION'
                );

            } catch (\Exception $e) {
                \Bitrix\Main\Diag\Logger::log(
                    "Failed to migrate recording {$recording['ID']}: {$e->getMessage()}",
                    'RECORDING_MIGRATION_ERROR'
                );
            }
        }

        \Bitrix\Main\Diag\Logger::log(
            "Recording migration completed: {$migratedCount} recordings migrated to S3",
            'RECORDING_MIGRATION'
        );
    }

    private function downloadFromTwilio($recordingSid) {
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        // Get recording metadata
        $recording = $twilioClient->recordings($recordingSid)->fetch();

        // Download recording content
        $recordingUrl = "https://api.twilio.com{$recording->uri}.mp3";

        $recordingContent = file_get_contents($recordingUrl, false, stream_context_create([
            'http' => [
                'header' => "Authorization: Basic " . base64_encode(TWILIO_SID . ':' . TWILIO_TOKEN)
            ]
        ]));

        return $recordingContent;
    }

    private function uploadToS3($recordingContent, $recordingMeta) {
        // S3-compatible client (works with Backblaze B2, Wasabi, etc.)
        $s3Client = new Aws\S3\S3Client([
            'version' => 'latest',
            'region' => S3_REGION,
            'endpoint' => S3_ENDPOINT,  // e.g., https://s3.us-west-002.backblazeb2.com
            'credentials' => [
                'key' => S3_ACCESS_KEY,
                'secret' => S3_SECRET_KEY
            ]
        ]);

        $fileName = 'recordings/' . date('Y/m/d', strtotime('-30 days')) . '/' . $recordingMeta['TWILIO_SID'] . '.mp3';

        $s3Client->putObject([
            'Bucket' => S3_BUCKET,
            'Key' => $fileName,
            'Body' => $recordingContent,
            'ContentType' => 'audio/mpeg',
            'ServerSideEncryption' => 'AES256',
            'Metadata' => [
                'TwilioSid' => $recordingMeta['TWILIO_SID'],
                'Duration' => $recordingMeta['DURATION'],
                'EntityType' => $recordingMeta['ENTITY_TYPE'],
                'EntityId' => $recordingMeta['ENTITY_ID'],
                'MigratedDate' => date('Y-m-d H:i:s')
            ]
        ]);

        // Generate permanent URL (or presigned URL with long expiry)
        $s3Url = $s3Client->getObjectUrl(S3_BUCKET, $fileName);

        return $s3Url;
    }

    private function deleteFromTwilio($recordingSid) {
        try {
            $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);
            $twilioClient->recordings($recordingSid)->delete();

            \Bitrix\Main\Diag\Logger::log(
                "Deleted recording {$recordingSid} from Twilio after migration",
                'RECORDING_MIGRATION'
            );
        } catch (\Exception $e) {
            \Bitrix\Main\Diag\Logger::log(
                "Failed to delete Twilio recording {$recordingSid}: {$e->getMessage()}",
                'RECORDING_MIGRATION_ERROR'
            );
        }
    }
}
```

#### 3.3.3 Transparent URL Handling
```php
// Bitrix24 динамічно повертає актуальну URL (Twilio або S3)
class RecordingUrlResolver {
    public static function getPlayableUrl($recordingId) {
        global $DB;

        $recording = $DB->Query("
            SELECT RECORDING_URL, STORAGE_LOCATION, TWILIO_SID
            FROM b_crm_call_recording
            WHERE ID = {$recordingId}
        ")->Fetch();

        if (!$recording) {
            throw new \Exception('Recording not found');
        }

        // If already on S3, return S3 URL directly
        if ($recording['STORAGE_LOCATION'] === 'S3') {
            return $recording['RECORDING_URL'];
        }

        // If on Twilio, generate presigned URL with authentication
        if ($recording['STORAGE_LOCATION'] === 'TWILIO') {
            return $recording['RECORDING_URL'] . '?auth=' . $this->generateTwilioAuth($recording['TWILIO_SID']);
        }

        // Fallback
        return $recording['RECORDING_URL'];
    }

    private function generateTwilioAuth($recordingSid) {
        // Generate temporary auth token for Twilio recording access
        return base64_encode(TWILIO_SID . ':' . TWILIO_TOKEN);
    }
}
```

```javascript
            data: {
                recordingId: this.recordingId,
                userId: BX.message('USER_ID')
            }
        });
    }
```

```php
// Backend: /rest/supervisor/listen
class SupervisorMonitoring {
    public function listenAction($callSid, $supervisorUserId) {
        // Get active call from Twilio
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);
        $call = $twilioClient->calls($callSid)->fetch();

        if ($call->status !== 'in-progress') {
            throw new \Exception('Call is not active');
        }

        // Create TwiML for supervisor (listen only)
        $twiml = new VoiceResponse();
        $dial = $twiml->dial();
        $dial->conference($call->sid, [
            'muted' => true,  // Supervisor cannot be heard
            'beep' => false,
            'startConferenceOnEnter' => false,
            'endConferenceOnExit' => false
        ]);

        // Save TwiML to temporary URL
        $twimlUrl = $this->saveTwimlToUrl($twiml);

        // Call supervisor
        $supervisorPhone = $this->getSupervisorPhone($supervisorUserId);
        $supervisorCall = $twilioClient->calls->create(
            $supervisorPhone,
            TWILIO_NUMBER,
            ['url' => $twimlUrl]
        );

        // Log monitoring activity
        $this->logMonitoring([
            'SUPERVISOR_USER_ID' => $supervisorUserId,
            'CALL_SID' => $callSid,
            'ACTION' => 'LISTEN',
            'DATE_START' => new DateTime()
        ]);

        return [
            'success' => true,
            'supervisorCallSid' => $supervisorCall->sid
        ];
    }
```

```php
// Twilio Insights API: get call quality metrics
class CallQualityMetrics {
    public function getCallQuality($callSid) {
        $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);

        // Fetch call with quality metrics
        $call = $twilioClient->calls($callSid)
            ->fetch([
                'Include' => 'connectivity,carrier-edge'
            ]);

        // Get detailed metrics from Insights API
        $insights = $twilioClient->insights->v1->calls($callSid)->fetch();

        $metrics = [
            'mos_score' => null,
            'jitter' => null,
            'latency' => null,
            'packet_loss' => null
        ];

        // Parse metrics from insights
        if ($insights->metrics) {
            $metrics['mos_score'] = $insights->metrics->inbound->mos ?? null;
            $metrics['jitter'] = $insights->metrics->inbound->jitter ?? null;
            $metrics['latency'] = $insights->metrics->inbound->rtt ?? null;
            $metrics['packet_loss'] = $insights->metrics->inbound->packetLoss ?? null;
        }

        return $metrics;
    }

    public function interpretMosScore($mos) {
        if ($mos === null) return 'Unknown';
        if ($mos >= 4.0) return 'Excellent';
        if ($mos >= 3.5) return 'Good';
        if ($mos >= 3.0) return 'Fair';
        if ($mos >= 2.5) return 'Poor';
        return 'Bad';
    }
}
```

**MOS (Mean Opinion Score):**
- 5.0: Perfect (uncompressed audio)
- 4.0-4.5: Excellent (toll quality)
- 3.5-4.0: Good (acceptable for VoIP)
- 3.0-3.5: Fair (noticeable degradation)
- < 3.0: Poor (unacceptable)

## 6. Compliance & Legal

### 6.1 Retention Policy
```php
// Cron job: delete recordings older than 2 years
class RecordingRetentionPolicy {
    public function execute() {
        $cutoffDate = date('Y-m-d', strtotime('-2 years'));

        global $DB;

        // Get expired recordings
        $recordings = $DB->Query("
            SELECT ID, RECORDING_URL, TWILIO_SID
            FROM b_crm_call_recording
            WHERE DATE_CREATE < '{$cutoffDate}'
        ");

        $deletedCount = 0;

        while ($recording = $recordings->Fetch()) {
            // Delete from S3
            $this->deleteFromS3($recording['RECORDING_URL']);

            // Delete from Twilio (if still there)
            $this->deleteFromTwilio($recording['TWILIO_SID']);

            // Delete from database
            $DB->Query("DELETE FROM b_crm_call_recording WHERE ID = {$recording['ID']}");

            $deletedCount++;
        }

        \Bitrix\Main\Diag\Logger::log(
            "Recording retention cleanup: {$deletedCount} recordings deleted",
            'RECORDING_RETENTION'
        );
    }

    private function deleteFromS3($s3Url) {
        $s3Client = new Aws\S3\S3Client([...]);

        // Parse S3 URL to get bucket and key
        $parsedUrl = parse_url($s3Url);
        $path = ltrim($parsedUrl['path'], '/');

        $s3Client->deleteObject([
            'Bucket' => AWS_S3_BUCKET,
            'Key' => $path
        ]);
    }

    private function deleteFromTwilio($recordingSid) {
        try {
            $twilioClient = new Twilio\Rest\Client(TWILIO_SID, TWILIO_TOKEN);
            $twilioClient->recordings($recordingSid)->delete();
        } catch (\Exception $e) {
            // Recording may already be deleted or not exist
            \Bitrix\Main\Diag\Logger::log(
                "Failed to delete Twilio recording {$recordingSid}: {$e->getMessage()}",
                'RECORDING_RETENTION_ERROR'
            );
        }
    }
}
```
