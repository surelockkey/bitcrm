# Story 12.01: Data Import to Bitrix24

**Epic:** [EPIC 12: Communications & Corporate Messenger Migration](../epics/EPIC-12-Communications.md)

---

## User Story

**As a** Shift Manager
**I want** to import all filtered communication history from Flock and WhatsApp into Bitrix24 messenger
**So that** all employees have access to historical conversations and can search past communications

---

## Scope Note

**Client Responsibility:** The client team will handle data export from Flock and WhatsApp internally. This Story covers only the integrator's responsibilities.

**Integrator Receives:**
- Pre-exported data files (JSON/CSV format) from client
- Filtered list of chats to migrate (client has already removed personal/non-business chats)
- File attachments organized by chat

**Integrator Delivers:**
- Data validation and format checking
- Import of messages, chats, and files into Bitrix24
- User mapping and verification
- Import validation reports

---

## Acceptance Criteria

### 1. Data Format Specification

**Given** the client will export data themselves
**When** defining the data format specification
**Then** the system must support the following structure:

```
expected_format/
```

### 2. Data Validation Tool

**Given** the client has exported data from Flock/WhatsApp

**Then** a validation script must check:

- ✅ All required JSON files present (users.json, chats.json, messages.json)
- ✅ JSON files are valid and parseable
- ✅ Required fields present in each record:
  - users.json: `id`, `name`, `email`
  - chats.json: `chat_id`, `name`, `type`, `members`
  - messages.json: `message_id`, `chat_id`, `sender_id`, `text`, `timestamp`
- ✅ All referenced files in `attachments` exist in `files/` directory
- ✅ All `sender_id` values in messages.json exist in users.json
- ✅ All `members` in chats.json exist in users.json
- ✅ No duplicate `message_id`, `chat_id`, or user `id` values
- ✅ Timestamps are in valid ISO 8601 format

### 5. Message History Import

**Given** the messages from messages.json
**When** importing message history
**Then** the system must:

- ✅ Import messages in chronological order (oldest first)
- ✅ Preserve original timestamp from `timestamp` field
- ✅ Set correct author based on mapped `sender_id`
- ✅ Handle messages with attachments (link to uploaded files)
- ✅ Batch import in chunks (e.g., 100 messages at a time) to avoid timeouts
- ✅ Create progress indicator for large imports

**Message Import Requirements:**

**Timestamp Preservation:**
- Bitrix24 may not support backdating messages via standard API
- If not possible: Document limitation and import with current timestamp
- Add original timestamp to message text (e.g., "[Original: 2024-03-20 14:35] Message text")

**Author Attribution:**
- Messages must show original sender's name
- Use Bitrix24 "post on behalf of" feature if available
- Alternative: Add sender name prefix (e.g., "[John Smith]: Message text")

**Handling Large Imports:**
```php
$batchSize = 100;
$messageChunks = array_chunk($messages, $batchSize);

foreach ($messageChunks as $index => $chunk) {

```

**Given** an error occurs during import
**When** the import process encounters a failure
**Then** the system must:

- ✅ Log all errors with context (chat_id, message_id, error details)
- ✅ Continue import process (don't stop on single error)
- ✅ Provide option to retry failed imports
- ✅ Support rollback of import (delete all imported data)
- ✅ Maintain backup of original export files

**Rollback Capability:**
```php
function rollbackImport($importSessionId) {
    // Get all entities created in this import session
    $importedChats = getImportedChats($importSessionId);
    $importedMessages = getImportedMessages($importSessionId);
    $importedFiles = getImportedFiles($importSessionId);

    // Delete in reverse order
    foreach ($importedMessages as $message) {
        deleteBitrix24Message($message['BITRIX24_ID']);
    }

    foreach ($importedFiles as $file) {
        deleteBitrix24File($file['BITRIX24_ID']);
    }

    foreach ($importedChats as $chat) {
        deleteBitrix24Chat($chat['BITRIX24_ID']);
    }

    // Log rollback
    logImportRollback($importSessionId, "Rolled back by admin");
}
```

## Technical Implementation

### Data Validation Script

**Script:** `validate_messenger_export.py`

```python
#!/usr/bin/env python3
"""
Validates exported Flock/WhatsApp data before import to Bitrix24
"""

import json
import os
import sys
from datetime import datetime
from pathlib import Path

class MessengerDataValidator:
    def __init__(self, export_dir):
        self.export_dir = Path(export_dir)
        self.errors = []
        self.warnings = []

    def validate(self):
        print("VALIDATION REPORT")
        print("=================\n")

        # Check file structure
        self.validate_file_structure()

        # Load JSON files
        users = self.load_json('users.json')
        chats = self.load_json('chats.json')
        messages = self.load_json('messages.json')

        if not (users and chats and messages):
            print("\n❌ ERRORS: Cannot proceed - JSON files invalid")
            return False

        # Validate data integrity
        self.validate_users(users)
        self.validate_chats(chats, users)
        self.validate_messages(messages, chats, users)
        self.validate_files(messages)

        # Print summary
        self.print_summary()

        return len(self.errors) == 0

    def validate_file_structure(self):
        print("✅ File Structure:")

        required_files = ['users.json', 'chats.json', 'messages.json']
        for filename in required_files:
            filepath = self.export_dir / filename
            if filepath.exists():
                size = filepath.stat().st_size
                print(f"   - {filename}: Found ({size:,} bytes)")
            else:
                self.errors.append(f"Missing required file: {filename}")
                print(f"   ❌ {filename}: NOT FOUND")

        files_dir = self.export_dir / 'files'
        if files_dir.exists():
            file_count = len(list(files_dir.rglob('*.*')))
            print(f"   - files/: Found ({file_count} files)")
        else:
            self.warnings.append("files/ directory not found")
            print(f"   ⚠️  files/: NOT FOUND (no attachments?)")

        print()

    def load_json(self, filename):
        filepath = self.export_dir / filename
        try:
            with open(filepath, 'r', encoding='utf-8') as f:
                data = json.load(f)
                print(f"✅ {filename}: Valid JSON ({len(data)} records)")
                return data
        except json.JSONDecodeError as e:
            self.errors.append(f"{filename}: Invalid JSON - {e}")
            print(f"❌ {filename}: INVALID JSON")
            return None
        except Exception as e:
            self.errors.append(f"{filename}: Error reading file - {e}")
            print(f"❌ {filename}: ERROR - {e}")
            return None

    def validate_users(self, users):
        print("\n✅ User Validation:")

        user_ids = set()
        no_email = 0

        for user in users:
            # Check required fields
            if 'id' not in user:
                self.errors.append(f"User missing 'id' field: {user}")
                continue

            if user['id'] in user_ids:
                self.errors.append(f"Duplicate user id: {user['id']}")
            user_ids.add(user['id'])

            if 'name' not in user:
                self.errors.append(f"User {user['id']} missing 'name' field")

            if 'email' not in user or not user['email']:
                no_email += 1

        print(f"   - Total users: {len(users)}")
        print(f"   - Unique IDs: {len(user_ids)}")

        if no_email > 0:
            self.warnings.append(f"{no_email} users have no email (will need manual mapping)")
            print(f"   ⚠️  {no_email} users without email")

    def validate_chats(self, chats, users):
        print("\n✅ Chat Validation:")

        user_ids = {u['id'] for u in users}
        chat_ids = set()
        invalid_members = 0

        for chat in chats:
            if 'chat_id' not in chat:
                self.errors.append(f"Chat missing 'chat_id' field: {chat}")
                continue

            if chat['chat_id'] in chat_ids:
                self.errors.append(f"Duplicate chat_id: {chat['chat_id']}")
            chat_ids.add(chat['chat_id'])

            # Check members exist in users
            if 'members' in chat:
                for member_id in chat['members']:
                    if member_id not in user_ids:
                        self.errors.append(f"Chat {chat['chat_id']} has invalid member: {member_id}")
                        invalid_members += 1

        print(f"   - Total chats: {len(chats)}")
        print(f"   - Group chats: {sum(1 for c in chats if c.get('type') == 'group')}")
        print(f"   - Private chats: {sum(1 for c in chats if c.get('type') == 'private')}")

        if invalid_members > 0:
            print(f"   ❌ {invalid_members} invalid member references")

    def validate_messages(self, messages, chats, users):
        print("\n✅ Message Validation:")

        user_ids = {u['id'] for u in users}
        chat_ids = {c['chat_id'] for c in chats}

        invalid_sender = 0
        invalid_chat = 0
        invalid_timestamp = 0

        for msg in messages:
            # Check sender exists
            if 'sender_id' in msg and msg['sender_id'] not in user_ids:
                invalid_sender += 1

            # Check chat exists
            if 'chat_id' in msg and msg['chat_id'] not in chat_ids:
                invalid_chat += 1

            # Check timestamp format
            if 'timestamp' in msg:
                try:
                    datetime.fromisoformat(msg['timestamp'].replace('Z', '+00:00'))
                except:
                    invalid_timestamp += 1

        print(f"   - Total messages: {len(messages)}")

        if invalid_sender > 0:
            self.errors.append(f"{invalid_sender} messages have invalid sender_id")
            print(f"   ❌ {invalid_sender} messages with invalid sender")

        if invalid_chat > 0:
            self.errors.append(f"{invalid_chat} messages have invalid chat_id")
            print(f"   ❌ {invalid_chat} messages with invalid chat")

        if invalid_timestamp > 0:
            self.warnings.append(f"{invalid_timestamp} messages have invalid timestamp")
            print(f"   ⚠️  {invalid_timestamp} messages with invalid timestamp")

    def validate_files(self, messages):
        print("\n✅ File Validation:")

        referenced_files = set()
        for msg in messages:
            if 'attachments' in msg:
                for attachment in msg['attachments']:
                    referenced_files.add(attachment)

        print(f"   - Files referenced in messages: {len(referenced_files)}")

        missing_files = 0
        for file_path in referenced_files:
            full_path = self.export_dir / file_path
            if not full_path.exists():
                missing_files += 1
                self.errors.append(f"Referenced file not found: {file_path}")

        if missing_files > 0:
            print(f"   ❌ {missing_files} referenced files not found")
        else:
            print(f"   ✅ All referenced files exist")

    def print_summary(self):
        print("\n" + "="*50)

        if self.warnings:
            print(f"\n⚠️  WARNINGS ({len(self.warnings)}):")
            for warning in self.warnings:
                print(f"   - {warning}")

        if self.errors:
            print(f"\n❌ ERRORS ({len(self.errors)}):")
            for error in self.errors[:10]:  # Show first 10
                print(f"   - {error}")
            if len(self.errors) > 10:
                print(f"   ... and {len(self.errors) - 10} more errors")

        print("\n" + "="*50)

        if len(self.errors) == 0:
            print("\n✅ READY FOR IMPORT: YES")
        else:
            print("\n❌ READY FOR IMPORT: NO - Fix errors first")

if __name__ == '__main__':
    if len(sys.argv) != 2:
        print("Usage: python validate_messenger_export.py <export_directory>")
        sys.exit(1)

    validator = MessengerDataValidator(sys.argv[1])
    success = validator.validate()

    sys.exit(0 if success else 1)
```

### Import Script (PHP for Bitrix24)

**Script:** `import_messenger_data.php`

```php
<?php
require_once($_SERVER["DOCUMENT_ROOT"]."/bitrix/modules/main/include/prolog_before.php");

use Bitrix\Main\Loader;
use Bitrix\Im\V2\Chat;
use Bitrix\Im\V2\Message;
use Bitrix\Disk\Driver;

Loader::includeModule('im');
Loader::includeModule('disk');

class MessengerImporter
{
    private $exportDir;
    private $importSessionId;
    private $userMapping = [];
    private $chatMapping = [];
    private $stats = [
        'users_mapped' => 0,
        'chats_created' => 0,
        'messages_imported' => 0,
        'files_uploaded' => 0,
        'errors' => []
    ];

    public function __construct($exportDir)
    {
        $this->exportDir = $exportDir;
        $this->importSessionId = uniqid('import_', true);
    }

    public function import()
    {
        echo "Starting import session: {$this->importSessionId}\n\n";

        // Load data
        $users = $this->loadJson('users.json');
        $chats = $this->loadJson('chats.json');
        $messages = $this->loadJson('messages.json');

        // Step 1: Map users
        echo "Step 1: Mapping users...\n";
        $this->mapUsers($users);

        // Step 2: Create chats
        echo "\nStep 2: Creating chats...\n";
        $this->createChats($chats);

        // Step 3: Import messages
        echo "\nStep 3: Importing messages...\n";
        $this->importMessages($messages);

        // Step 4: Generate report
        echo "\nStep 4: Generating report...\n";
        $this->generateReport();
    }

    private function loadJson($filename)
    {
        $filepath = $this->exportDir . '/' . $filename;
        $content = file_get_contents($filepath);
        return json_decode($content, true);
    }

    private function mapUsers($users)
    {
        foreach ($users as $user) {
            $email = $user['email'] ?? null;

            if (!$email) {
                $this->stats['errors'][] = "User {$user['id']} has no email - cannot map";
                continue;
            }

            // Find Bitrix24 user by email
            $rsUser = \CUser::GetList(
                ($by = 'email'),
                ($order = 'asc'),
                ['EMAIL' => $email],
                ['SELECT' => ['ID', 'EMAIL', 'NAME', 'LAST_NAME']]
            );

            if ($bitrixUser = $rsUser->Fetch()) {
                $this->userMapping[$user['id']] = $bitrixUser['ID'];
                $this->stats['users_mapped']++;
                echo "  ✅ Mapped {$user['name']} ({$email}) → Bitrix24 User #{$bitrixUser['ID']}\n";
            } else {
                $this->stats['errors'][] = "No Bitrix24 user found for email: {$email}";
                echo "  ❌ Cannot map {$user['name']} ({$email}) - user not found\n";
            }
        }

        echo "\n  Summary: {$this->stats['users_mapped']} users mapped\n";
    }

    private function createChats($chats)
    {
        foreach ($chats as $chat) {
            if ($chat['type'] !== 'group') {
                continue; // Skip private chats
            }

            // Map members
            $members = [];
            foreach ($chat['members'] as $memberId) {
                if (isset($this->userMapping[$memberId])) {
                    $members[] = $this->userMapping[$memberId];
                }
            }

            if (empty($members)) {
                $this->stats['errors'][] = "Chat {$chat['chat_id']} has no valid members";
                continue;
            }

            // Create chat
            try {
                $chatResult = Chat\ChatFactory::getInstance()->addChat([
                    'TYPE' => Chat::IM_TYPE_OPEN,
                    'TITLE' => $chat['name'],
                    'DESCRIPTION' => "Migrated from {$chat['source']}",
                    'USERS' => $members,
                    'AUTHOR_ID' => $members[0], // First member as creator
                ]);

                if ($chatResult->isSuccess()) {
                    $bitrixChatId = $chatResult->getResult()['CHAT_ID'];
                    $this->chatMapping[$chat['chat_id']] = $bitrixChatId;
                    $this->stats['chats_created']++;

                    echo "  ✅ Created chat: {$chat['name']} (ID: {$bitrixChatId})\n";
                } else {
                    $errors = implode(', ', $chatResult->getErrorMessages());
                    $this->stats['errors'][] = "Failed to create chat {$chat['chat_id']}: {$errors}";
                    echo "  ❌ Failed to create chat: {$chat['name']}\n";
                }
            } catch (\Exception $e) {
                $this->stats['errors'][] = "Exception creating chat {$chat['chat_id']}: {$e->getMessage()}";
                echo "  ❌ Exception: {$e->getMessage()}\n";
            }
        }

        echo "\n  Summary: {$this->stats['chats_created']} chats created\n";
    }

    private function importMessages($messages)
    {
        $batchSize = 100;
        $totalMessages = count($messages);
        $batches = array_chunk($messages, $batchSize);

        echo "  Importing {$totalMessages} messages in " . count($batches) . " batches...\n";

        foreach ($batches as $index => $batch) {
            foreach ($batch as $message) {
                $this->importMessage($message);
            }

            $progress = (($index + 1) / count($batches)) * 100;
            echo "  Progress: " . round($progress, 1) . "%\n";

            sleep(1); // Throttle to avoid API limits
        }

        echo "\n  Summary: {$this->stats['messages_imported']} messages imported\n";
    }

    private function importMessage($message)
    {
        // Get Bitrix24 chat ID
        $bitrixChatId = $this->chatMapping[$message['chat_id']] ?? null;
        if (!$bitrixChatId) {
            $this->stats['errors'][] = "Message {$message['message_id']}: Chat not found";
            return;
        }

        // Get Bitrix24 user ID
        $bitrixUserId = $this->userMapping[$message['sender_id']] ?? null;
        if (!$bitrixUserId) {
            $this->stats['errors'][] = "Message {$message['message_id']}: Sender not found";
            return;
        }

        // Prepare message text with original timestamp
        $originalTimestamp = date('Y-m-d H:i:s', strtotime($message['timestamp']));
        $messageText = "[Migrated message from {$originalTimestamp}]\n" . $message['text'];

        try {
            // Send message
            $messageResult = \CIMMessenger::Add([
                'CHAT_ID' => $bitrixChatId,
                'FROM_USER_ID' => $bitrixUserId,
                'MESSAGE' => $messageText,
            ]);

            if ($messageResult) {
                $this->stats['messages_imported']++;

                // Handle attachments
                if (!empty($message['attachments'])) {
                    $this->uploadAttachments($messageResult, $message['attachments']);
                }
            } else {
                $this->stats['errors'][] = "Failed to import message {$message['message_id']}";
            }
        } catch (\Exception $e) {
            $this->stats['errors'][] = "Exception importing message {$message['message_id']}: {$e->getMessage()}";
        }
    }

        foreach ($attachments as $filePath) {

            }
```

## Risks & Mitigation

| Risk | Impact | Mitigation Strategy |
|------|--------|---------------------|
| **Invalid/incomplete data from client** | High - blocks import | - Provide clear data format specification<br>- Provide validation script to client<br>- Validate data before import<br>- Clear error messages |
| **Timestamp preservation not possible** | Medium - loses context | - Research Bitrix24 API capabilities early<br>- Document limitation if not possible<br>- Alternative: add timestamp to message text<br>- Set client expectations |
| **User mapping failures** | High - orphaned messages | - Use email as primary mapping key<br>- Generate mapping report before import<br>- Identify unmapped users early<br>- Manual review process for edge cases |
| **Large import timeouts** | Medium - incomplete import | - Batch processing (100 messages at a time)<br>- Progress tracking and resume capability<br>- Throttle API calls (1 second delay)<br>- Run import during off-hours |
| **File upload failures** | Medium - missing attachments | - Verify all files exist before import<br>- Log upload errors separately<br>- Retry failed uploads<br>- Acceptable if some non-critical files missing |
